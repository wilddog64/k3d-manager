# Bugfix: v1.6.3 — acg-up EXIT trap kills ArgoCD port-forward LaunchAgent

**Branch:** `k3d-manager-v1.6.3`
**Files:** `bin/acg-up`

---

## Problem

After a fresh `acg-up` (or failed `acg-up`), the ArgoCD smoke check fails with
`Connection refused` on port 8080. The ArgoCD port-forward LaunchAgent is not
running even though step 4b installed it.

**Root cause:** `_acg_up_cleanup` (registered as `trap EXIT`) calls
`launchctl unload argocd-port-forward` unconditionally on every exit — including
successful completion and mid-run failures. This undoes the step 4b install.

---

## Reproduction

```
bin/acg-up   # let it run past step 4b, then fail at step 10b
# _acg_up_cleanup fires on EXIT, unloads com.k3d-manager.argocd-port-forward
curl http://localhost:8080/healthz  # → Connection refused
```

---

## Fix

### Change 1 — `bin/acg-up`: remove ArgoCD unload from `_acg_up_cleanup`

The ArgoCD port-forward LaunchAgent is a **persistent** service installed during
step 4b. It must survive the acg-up process exiting. Only ephemeral PID-file
processes and the Cloudflare tunnel belong in the cleanup trap.

**Exact old block (lines 105–115):**

```bash
  _warn "[acg-up] failed (exit ${_exit_code}) — cleaning up local processes..."
  for _pid_file in vault-pf frontend-pf acg-prom-pf; do
    local _f="${_ACG_STATE_DIR}/run/${_pid_file}.pid"
    if [[ -f "${_f}" ]]; then
      kill "$(cat "${_f}")" 2>/dev/null || true
      rm -f "${_f}"
    fi
  done
  launchctl unload "${HOME}/Library/LaunchAgents/com.k3d-manager.argocd-port-forward.plist" 2>/dev/null || true
  launchctl bootout "gui/$(id -u)" "${HOME}/Library/LaunchAgents/com.k3d-manager.cloudflare-tunnel.plist" 2>/dev/null || true
}
trap _acg_up_cleanup EXIT
```

**Exact new block:**

```bash
  _warn "[acg-up] failed (exit ${_exit_code}) — cleaning up local processes..."
  for _pid_file in vault-pf frontend-pf acg-prom-pf; do
    local _f="${_ACG_STATE_DIR}/run/${_pid_file}.pid"
    if [[ -f "${_f}" ]]; then
      kill "$(cat "${_f}")" 2>/dev/null || true
      rm -f "${_f}"
    fi
  done
  launchctl bootout "gui/$(id -u)" "${HOME}/Library/LaunchAgents/com.k3d-manager.cloudflare-tunnel.plist" 2>/dev/null || true
}
trap _acg_up_cleanup EXIT
```
