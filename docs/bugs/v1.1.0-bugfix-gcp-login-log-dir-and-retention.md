## Problem

The Playwright log file is written to `~/.local/share/k3d-manager/gcp_login_pw.log` (flat,
single file, truncated on every run). Two issues:

1. **Wrong location** — logs should live under `~/.local/share/k3d-manager/logs/` for
   organizational clarity and to allow future log types alongside it.
2. **No retention** — the single file is overwritten each run with no history. Timestamped
   files with a 7-day retention policy (adjustable via `GCP_LOG_RETENTION_DAYS`) preserve
   enough history for debugging without unbounded disk growth.

---

## Fix

### Change 1 — log path, directory, and retention

**Exact old block (lines 188–190):**

```bash
  local _pw_log="${HOME}/.local/share/k3d-manager/gcp_login_pw.log"
  mkdir -p "${HOME}/.local/share/k3d-manager"
  : > "${_pw_log}"
```

**Exact new block:**

```bash
  local _log_dir="${HOME}/.local/share/k3d-manager/logs"
  local _log_retention="${GCP_LOG_RETENTION_DAYS:-7}"
  local _pw_log="${_log_dir}/gcp_login_pw_$(date +%Y%m%dT%H%M%S).log"
  mkdir -p "${_log_dir}"
  find "${_log_dir}" -name "gcp_login_pw_*.log" -mtime +"${_log_retention}" -delete 2>/dev/null || true
```
