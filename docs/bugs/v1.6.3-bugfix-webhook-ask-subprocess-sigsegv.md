# Bugfix: v1.6.3 — `/ask claude` and `/ask codex` crash Python with SIGSEGV (NEF atfork)

**Branch:** `k3d-manager-v1.6.3`
**Files:** `bin/k3dm-webhook`

---

## Problem

Running `/ask claude <question>` or `/ask codex <question>` from Slack causes Python to crash
with `EXC_BAD_ACCESS (SIGSEGV)` in `grp.cpython-313-darwin.so`. The crash appears in macOS
Problem Reporter as "Python quit unexpectedly" in the `com.k3d-manager.webhook` coalition.

**Root cause:** `_run_cluster_ask` dispatches claude and codex via `subprocess.run(cmd,
capture_output=True, ...)` at line 2023. `capture_output=True` internally calls `os.fork()`.
On macOS 26 (Sequoia), the Network Extension Framework registers an atfork handler that is
not safe to run in the child process — it reads freed memory and triggers SIGSEGV. This is
the same class of bug that was previously fixed for Gemini in `_call_gemini` (switched to
`os.posix_spawn` + temp-file output).

---

## Fix

### Change 1 — `bin/k3dm-webhook`: add `_posix_spawn_capture` helper

Add a new helper immediately after `_call_gemini` (after line 803, before `_analyze_stall`).
This helper wraps `os.posix_spawn` + temp-file capture with a wait loop and timeout — the
same pattern as `_call_gemini` but parameterized for arbitrary commands and timeouts.

**Insert after the closing `except Exception as exc:` block of `_call_gemini` (after line 803),
before `def _analyze_stall`:**

```python
def _posix_spawn_capture(cmd, timeout, cwd=None, env=None):
    """Run cmd via os.posix_spawn (no fork) and return stdout+stderr as a string.

    Uses posix_spawn(2) which does NOT trigger macOS NEF atfork handlers.
    cwd is handled by wrapping cmd in a bash -c cd && ... invocation.
    Returns (output_text, timed_out_bool).
    """
    import shlex as _shlex, shutil as _shutil, tempfile as _tmp, time as _time
    _env = dict(os.environ) if env is None else dict(env)
    exe = cmd[0]
    if not os.path.isabs(exe):
        exe = _shutil.which(exe) or exe
    if cwd:
        cmd_str = " ".join(_shlex.quote(str(c)) for c in cmd)
        spawn_cmd = ["/bin/bash", "-c", f"cd {_shlex.quote(str(cwd))} && {cmd_str}"]
        spawn_exe = "/bin/bash"
    else:
        spawn_cmd = list(cmd)
        spawn_exe = exe
    try:
        with _tmp.NamedTemporaryFile(
            prefix="k3dm-ask-", suffix=".out", delete=False, mode="w"
        ) as _f:
            tmp_path = _f.name
        file_actions = [
            (os.POSIX_SPAWN_OPEN, 1, tmp_path, os.O_WRONLY | os.O_CREAT | os.O_TRUNC, 0o600),
            (os.POSIX_SPAWN_DUP2, 1, 2),
        ]
        child_pid = os.posix_spawn(spawn_exe, spawn_cmd, _env, file_actions=file_actions, setsid=True)
        deadline = _time.monotonic() + timeout
        while True:
            if _time.monotonic() > deadline:
                try:
                    os.kill(child_pid, 9)
                    os.waitpid(child_pid, 0)
                except OSError:
                    pass
                return Path(tmp_path).read_text(errors="replace").strip(), True
            try:
                done_pid, _ = os.waitpid(child_pid, os.WNOHANG)
                if done_pid != 0:
                    break
            except ChildProcessError:
                break
            _time.sleep(0.5)
        output = Path(tmp_path).read_text(errors="replace").strip()
        Path(tmp_path).unlink(missing_ok=True)
        return output, False
    except Exception as exc:
        return f"spawn error: {exc}", False
```
