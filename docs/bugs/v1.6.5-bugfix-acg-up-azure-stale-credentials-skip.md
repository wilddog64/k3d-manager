# Bug: acg-up Step 1 skips Azure extraction on stale sandbox credentials — AZ_RESOURCE_GROUP left empty

**Date:** 2026-06-11
**Branch:** k3d-manager-v1.6.5
**File:** `bin/acg-up`

---

## Symptom

`make up CLUSTER_PROVIDER=k3s-azure` prints:

```
INFO: [acg-up] Azure credentials already valid — skipping Playwright extraction
ERROR: [k3s-azure] Could not determine resource group — set AZ_RESOURCE_GROUP
```

Step 2 fails immediately with `AZ_RESOURCE_GROUP` unset.

---

## Root Cause

Step 1 fast-path (lines 186–194) only checks `_az_ok` — which returns 0 if `az account show`
succeeds. Stale credentials from a **previous sandbox** satisfy `_az_ok` because the Azure
session token is still technically valid, but those credentials have no resource groups in the
current sandbox. `az group list` at line 196 returns an empty string, setting `AZ_RESOURCE_GROUP`
to empty. Step 2 (`deploy_cluster`) then fails.

---

## Fix

### Change 1 — `bin/acg-up`: fold resource-group check into the `_az_ok` fast-path

**Exact old block (lines 185–199):**

```bash
  k3s-azure)
    if _az_ok 2>/dev/null; then
      _info "[acg-up] Azure credentials already valid — skipping Playwright extraction"
    else
      _acg_cred_test_url="${sandbox_url:-${_ACG_SANDBOX_LIST_URL}}"
      _info "[acg-up] Extracting and validating Azure credentials..."
      _browser_launch
      "${REPO_ROOT}/scripts/lib/acg/bin/acg-credential-test" "${_acg_cred_test_url}" --provider azure || exit 1
      _info "[acg-up] Azure credentials extracted and validated."
    fi
    if [[ -z "${AZ_RESOURCE_GROUP:-}" ]]; then
      AZ_RESOURCE_GROUP=$(az group list --query '[0].name' -o tsv 2>/dev/null || true)
      export AZ_RESOURCE_GROUP
    fi
    ;;
```

**Exact new block:**

```bash
  k3s-azure)
    _az_existing_rg=$(az group list --query '[0].name' -o tsv 2>/dev/null || true)
    if _az_ok 2>/dev/null && [[ -n "${_az_existing_rg}" ]]; then
      _info "[acg-up] Azure credentials already valid — skipping Playwright extraction"
      AZ_RESOURCE_GROUP="${_az_existing_rg}"
      export AZ_RESOURCE_GROUP
    else
      _acg_cred_test_url="${sandbox_url:-${_ACG_SANDBOX_LIST_URL}}"
      _info "[acg-up] Extracting and validating Azure credentials..."
      _browser_launch
      "${REPO_ROOT}/scripts/lib/acg/bin/acg-credential-test" "${_acg_cred_test_url}" --provider azure || exit 1
      _info "[acg-up] Azure credentials extracted and validated."
      AZ_RESOURCE_GROUP=$(az group list --query '[0].name' -o tsv 2>/dev/null || true)
      export AZ_RESOURCE_GROUP
    fi
    [[ -n "${AZ_RESOURCE_GROUP:-}" ]] || { _error "[acg-up] Could not determine AZ_RESOURCE_GROUP after credential extraction — aborting"; exit 1; }
    ;;
```
