## Problem

`make up CLUSTER_PROVIDER=k3s-gcp` currently runs both `deploy_cluster` **and**
`gcp_provision_stack`, making GCP's `make up` a full-stack provisioner. AWS `make up`
is cluster-only (`bin/acg-up`). This breaks the shared lifecycle contract:

| Target | AWS | GCP (current) | GCP (desired) |
|--------|-----|----------------|----------------|
| `make up` | cluster only | cluster + full stack | cluster only |
| `make provision` | AWS SSM only | not wired | cluster + full stack |
| `make sync-apps` | app registration | app registration | app registration |

`make provision` has no `k3s-gcp` case, so there is no way to run `gcp_provision_stack`
without also re-running `deploy_cluster` embedded inside `make up`.

---

## Fix

### Change 1 — strip `gcp_provision_stack` from `make up`

**Exact old block (lines 17–20 of `Makefile`):**

```makefile
	  k3s-gcp) echo "[make] CLUSTER_PROVIDER=k3s-gcp — running deploy_cluster..."; \
	           CLUSTER_PROVIDER="$(CLUSTER_PROVIDER)" _GCP_SANDBOX_URL="$(URL)" scripts/k3d-manager deploy_cluster --confirm && \
	           echo "[make] CLUSTER_PROVIDER=k3s-gcp — running gcp_provision_stack..." && \
	           CLUSTER_PROVIDER="$(CLUSTER_PROVIDER)" GHCR_PAT="$(GHCR_PAT)" scripts/k3d-manager gcp_provision_stack ;; \
```

**Exact new block:**

```makefile
	  k3s-gcp) echo "[make] CLUSTER_PROVIDER=k3s-gcp — running deploy_cluster..."; \
	           CLUSTER_PROVIDER="$(CLUSTER_PROVIDER)" _GCP_SANDBOX_URL="$(URL)" scripts/k3d-manager deploy_cluster --confirm ;; \
```

---

### Change 2 — add `k3s-gcp` case to `make provision`

**Exact old block (lines 116–121 of `Makefile`):**

```makefile
provision:
	@case "$(CLUSTER_PROVIDER)" in \
	  *)       $(MAKE) --no-print-directory ssm; \
	           echo "[make] CLUSTER_PROVIDER=$(CLUSTER_PROVIDER) — running acg_provision..."; \
	           K3S_AWS_SSM_ENABLED=true scripts/k3d-manager acg_provision --confirm ;; \
	esac
```

**Exact new block:**

```makefile
provision:
	@case "$(CLUSTER_PROVIDER)" in \
	  k3s-gcp) echo "[make] CLUSTER_PROVIDER=k3s-gcp — provisioning cluster + full stack..."; \
	           $(MAKE) --no-print-directory up CLUSTER_PROVIDER=k3s-gcp URL="$(URL)" GHCR_PAT="$(GHCR_PAT)" && \
	           CLUSTER_PROVIDER="$(CLUSTER_PROVIDER)" GHCR_PAT="$(GHCR_PAT)" scripts/k3d-manager gcp_provision_stack ;; \
	  *)       $(MAKE) --no-print-directory ssm; \
	           echo "[make] CLUSTER_PROVIDER=$(CLUSTER_PROVIDER) — running acg_provision..."; \
	           K3S_AWS_SSM_ENABLED=true scripts/k3d-manager acg_provision --confirm ;; \
	esac
```
