# Bugfix: v1.4.11 — Keycloak LDAP federation missing group mapper

**Branch:** `k3d-manager-v1.4.11`
**Repos:** `shopping-cart-infra` (permanent fix), `k3d-manager` (`bin/acg-up` reconcile step)

---

## Problem

ArgoCD SSO users land on `policy.default: role:readonly` even though LDAP groups exist
and RBAC policy maps those groups to ArgoCD roles. Users can view apps but cannot sync —
`permission denied: applications, sync, shopping-cart/<app>`.

**Root cause:** The Keycloak LDAP federation (`ee968f21`) has no `group-ldap-mapper`
component. Keycloak syncs user attributes but never syncs LDAP group memberships, so
all users have zero Keycloak group memberships and the groups claim in the JWT is empty.
ArgoCD receives an empty `groups` array and falls through to the default readonly role.

---

## Reproduction

1. Log into ArgoCD via SSO (any LDAP user)
2. Try to sync an application
3. Error: `Unable to sync: permission denied: applications, sync, shopping-cart/<app>`
4. Confirm via Keycloak admin: GET `/admin/realms/shopping-cart/users/<sub>/groups` → `[]`

---

## Fix

### Part 1 — `shopping-cart-infra`: Add `group-ldap-mapper` to realm import

**File:** `identity/keycloak/keycloak-reconcile-hook-job.yaml`
**Branch:** `chore/add-group-ldap-mapper`

The file already has a `create_mapper_if_missing` helper (for user-attribute mappers) and a
`ldap_id` variable. Insert the group mapper block **after** the last `create_mapper_if_missing`
call (line 225) and **before** the existing `triggerFullSync` call (line 227).

**Exact old block (lines 225–229):**

```sh
            create_mapper_if_missing "modify date" "modifyTimestamp" "modifyTimestamp" "true" "false"

            /opt/keycloak/bin/kcadm.sh create \
              "user-storage/${ldap_id}/sync?action=triggerFullSync" \
              -r "${KC_REALM}" >/dev/null
```

**Exact new block:**

```sh
            create_mapper_if_missing "modify date" "modifyTimestamp" "modifyTimestamp" "true" "false"

            _gm_existing="$(
              /opt/keycloak/bin/kcadm.sh get components \
                -r "${KC_REALM}" \
                -q parent="${ldap_id}" \
                -q type=org.keycloak.storage.ldap.mappers.LDAPStorageMapper \
                -q name="group-mapper" \
                --fields id \
                2>/dev/null \
              | grep -c '"id"' || true
            )"
            if [ "${_gm_existing}" -gt 0 ]; then
              echo "LDAP group mapper already exists"
            else
              _gm_file="$(mktemp /tmp/keycloak-group-mapper-XXXXXX.json)"
              cat > "${_gm_file}" <<GMEOF
{"parentId":"${ldap_id}","providerType":"org.keycloak.storage.ldap.mappers.LDAPStorageMapper","providerId":"group-ldap-mapper","name":"group-mapper","config":{"groups.dn":["ou=groups,dc=shopping-cart,dc=local"],"group.name.ldap.attribute":["cn"],"group.object.classes":["groupOfNames"],"preserve.group.inheritance":["true"],"ignore.missing.groups":["false"],"membership.ldap.attribute":["member"],"membership.attribute.type":["DN"],"membership.user.ldap.attribute":["uid"],"mode":["READ_ONLY"],"user.roles.retrieve.strategy":["LOAD_GROUPS_BY_MEMBER_ATTRIBUTE"],"memberof.ldap.attribute":["memberOf"],"drop.non.existing.groups.during.sync":["false"]}}
GMEOF
              /opt/keycloak/bin/kcadm.sh create components -r "${KC_REALM}" -f "${_gm_file}" >/dev/null
              rm -f "${_gm_file}"
              echo "Created LDAP group mapper"
            fi
            _gm_id="$(
              /opt/keycloak/bin/kcadm.sh get components \
                -r "${KC_REALM}" \
                -q parent="${ldap_id}" \
                -q type=org.keycloak.storage.ldap.mappers.LDAPStorageMapper \
                -q name="group-mapper" \
                --fields id \
                2>/dev/null \
              | grep '"id"' | head -1 \
              | sed 's/.*"id" : "\([^"]*\)".*/\1/'
            )"
            if [ -n "${_gm_id}" ]; then
              /opt/keycloak/bin/kcadm.sh create \
                "user-storage/${ldap_id}/mappers/${_gm_id}/sync?direction=fedToKeycloak" \
                -r "${KC_REALM}" >/dev/null
              echo "LDAP group sync complete"
            fi

            /opt/keycloak/bin/kcadm.sh create \
              "user-storage/${ldap_id}/sync?action=triggerFullSync" \
              -r "${KC_REALM}" >/dev/null
```

**Exact new block — insert after line 659:**

```bash
  fi
fi

_info "[acg-up] Step 10d.7/14 — Reconciling Keycloak LDAP group mapper..."
if [[ -z "${_kc_pod:-}" ]] || [[ -z "${_ldap_component_id:-}" ]]; then
  _warn "[acg-up] Keycloak pod or LDAP component ID unavailable — skipping group mapper reconciliation"
else
  _gm_existing=$(kubectl exec -n identity --context k3d-k3d-cluster "${_kc_pod}" -- \
    sh -c "/opt/keycloak/bin/kcadm.sh get components -r shopping-cart \
      -q parent=${_ldap_component_id} \
      -q type=org.keycloak.storage.ldap.mappers.LDAPStorageMapper \
      -q name=group-mapper \
      --fields id 2>/dev/null | grep -c '\"id\"' || true" 2>/dev/null || echo "0")
  if [[ "${_gm_existing}" -gt 0 ]]; then
    _info "[acg-up] LDAP group mapper already exists"
  else
    _gm_json='{"parentId":"'"${_ldap_component_id}"'","providerType":"org.keycloak.storage.ldap.mappers.LDAPStorageMapper","providerId":"group-ldap-mapper","name":"group-mapper","config":{"groups.dn":["ou=groups,dc=shopping-cart,dc=local"],"group.name.ldap.attribute":["cn"],"group.object.classes":["groupOfNames"],"preserve.group.inheritance":["true"],"ignore.missing.groups":["false"],"membership.ldap.attribute":["member"],"membership.attribute.type":["DN"],"membership.user.ldap.attribute":["uid"],"mode":["READ_ONLY"],"user.roles.retrieve.strategy":["LOAD_GROUPS_BY_MEMBER_ATTRIBUTE"],"memberof.ldap.attribute":["memberOf"],"drop.non.existing.groups.during.sync":["false"]}}'
    kubectl exec -n identity --context k3d-k3d-cluster "${_kc_pod}" -- \
      sh -c "echo '${_gm_json}' > /tmp/gm.json && \
        /opt/keycloak/bin/kcadm.sh create components -r shopping-cart -f /tmp/gm.json >/dev/null && \
        rm -f /tmp/gm.json" 2>/dev/null && \
      _info "[acg-up] LDAP group mapper created" || \
      _warn "[acg-up] LDAP group mapper creation failed"
  fi
  _gm_id=$(kubectl exec -n identity --context k3d-k3d-cluster "${_kc_pod}" -- \
    sh -c "/opt/keycloak/bin/kcadm.sh get components -r shopping-cart \
      -q parent=${_ldap_component_id} \
      -q type=org.keycloak.storage.ldap.mappers.LDAPStorageMapper \
      -q name=group-mapper \
      --fields id 2>/dev/null | grep '\"id\"' | head -1 | sed 's/.*: \"//;s/\".*//' " 2>/dev/null || true)
  if [[ -n "${_gm_id}" ]]; then
    kubectl exec -n identity --context k3d-k3d-cluster "${_kc_pod}" -- \
      sh -c "/opt/keycloak/bin/kcadm.sh create \
        'user-storage/${_ldap_component_id}/mappers/${_gm_id}/sync?direction=fedToKeycloak' \
        -r shopping-cart >/dev/null 2>&1" 2>/dev/null && \
      _info "[acg-up] LDAP group sync complete" || \
      _warn "[acg-up] LDAP group sync failed"
  fi
fi
```
