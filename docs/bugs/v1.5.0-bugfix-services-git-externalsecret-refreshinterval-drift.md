# Bug: services-git apps permanently OutOfSync due to ExternalSecret refreshInterval normalization

**Branch:** `k3d-manager-v1.5.0`
**File:** `scripts/etc/argocd/applicationsets/services-git.yaml`
**Severity:** P3 ŌĆö cosmetic OutOfSync; apps are healthy but ArgoCD never reaches Synced

---

## Symptom

After `make up`, `shopping-cart-product-catalog` (and potentially any other app managed by
`services-git`) shows `OutOfSync` permanently:

```
cicd  shopping-cart-product-catalog  OutOfSync  Progressing
```

`argocd app diff` shows:

```
===== external-secrets.io/ExternalSecret shopping-cart-apps/product-catalog-secrets ======
118c118
<   refreshInterval: 24h0m0s
---
>   refreshInterval: 24h
```

The app has `selfHeal: true`, so ArgoCD repeatedly re-applies `24h` from git, Kubernetes
normalizes it to `24h0m0s`, and the next comparison detects drift again ŌĆö infinite loop.

---

## Root Cause

Kubernetes normalizes duration strings: `24h` ŌåÆ `24h0m0s` when stored. ArgoCD compares
the stored value against git, sees a difference, and marks the resource OutOfSync. The
`services-git` ApplicationSet template does not ignore this field, so every app with an
ExternalSecret using a plain hour value (e.g. `24h`) is permanently OutOfSync.

---

## Fix

Add `ignoreDifferences` for `ExternalSecret.spec.refreshInterval` to the `services-git`
ApplicationSet template. The existing template already has an `ignoreDifferences` block for
`payment-db-credentials` ŌĆö add the ExternalSecret entry alongside it.

**`scripts/etc/argocd/applicationsets/services-git.yaml` ŌĆö replace `ignoreDifferences` block with:**

```yaml
      ignoreDifferences:
        - group: ""
          kind: Secret
          name: payment-db-credentials
          namespace: shopping-cart-payment
          jsonPointers:
            - /data/username
            - /data/password
        - group: external-secrets.io
          kind: ExternalSecret
          jqPathExpressions:
            - .spec.refreshInterval
```

---

## Behaviour After Fix

| Scenario | Before | After |
|----------|--------|-------|
| ExternalSecret with `refreshInterval: 24h` | Permanent OutOfSync loop | Synced ŌĆö drift ignored |
| ExternalSecret with other fields changed in git | OutOfSync (expected) | OutOfSync (expected ŌĆö only refreshInterval is ignored) |
| payment-db-credentials secret drift | Ignored (unchanged) | Ignored (unchanged) |
