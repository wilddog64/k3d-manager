## Problem

Two bugs in `scripts/playwright/acg_credentials.js`:

### Bug 1 — CDP fallback ignores live Chrome session (breaks `make up` AWS and GCP)

When Chrome is running with valid Pluralsight cookies but no Pluralsight tab is open,
the CDP block sets `browserContext` only when `_cdpPsPage` is found. If no Pluralsight
tab is open, `browserContext` stays `undefined` → script closes the CDP connection →
falls back to `launchPersistentContext(AUTH_DIR)` — an isolated profile with no cookies
→ Pluralsight redirects to `/id` → 300s timeout.

The fix: whenever CDP connects and returns at least one context, use that context.
The Pluralsight tab check becomes informational only (log different messages).
The fallback to `launchPersistentContext` must only trigger when CDP connection fails
entirely (Chrome not running).

### Bug 2 — `process.exit(1)` in catch bypasses `finally` (causes SingletonLock)

`process.exit(1)` at line 428 terminates the Node.js process immediately, before
the `finally` block can close the browser context. This leaves `SingletonLock` in
`playwright-auth/`, blocking the next `make up` run with:

```
ERROR: browserType.launchPersistentContext: Failed to create a ProcessSingleton
```

The fix: replace `process.exit(1)` in the inner `catch` with `throw error` so the
`finally` block always runs to close the browser. The outer `Promise.race` `.catch`
at the bottom of the file already calls `process.exit(1)` after cleanup.

## Fix

### Change 1 — CDP block (lines 148–163)

**Exact old block:**

```js
      const _cdpContexts = _cdpBrowser.contexts();
      if (_cdpContexts.length > 0) {
        const _cdpContext = _cdpContexts[0];
        const _cdpPages = _cdpContext.pages();
        const _cdpPsPage = _cdpPages.find(p => {
          try { return new URL(p.url()).hostname.endsWith('.pluralsight.com'); } catch { return false; }
        });
        if (_cdpPsPage) {
          console.error('INFO: Found existing Pluralsight session via CDP — reusing existing Chrome instance.');
          browserContext = _cdpContext;
        }
      }
      if (!browserContext) {
        try { await _cdpBrowser.close(); } catch {}
        _cdpBrowser = null;
      }
```

**Exact new block:**

```js
      const _cdpContexts = _cdpBrowser.contexts();
      if (_cdpContexts.length > 0) {
        const _cdpContext = _cdpContexts[0];
        const _cdpPages = _cdpContext.pages();
        const _cdpPsPage = _cdpPages.find(p => {
          try { return new URL(p.url()).hostname.endsWith('.pluralsight.com'); } catch { return false; }
        });
        if (_cdpPsPage) {
          console.error('INFO: Found existing Pluralsight session via CDP — reusing existing Chrome instance.');
        } else {
          console.error('INFO: CDP connected — no Pluralsight tab open, will open one in existing Chrome.');
        }
        browserContext = _cdpContext;
      }
      if (!browserContext) {
        try { await _cdpBrowser.close(); } catch {}
        _cdpBrowser = null;
      }
```

### Change 2 — catch block (lines 426–428)

**Exact old block:**

```js
  } catch (error) {
    console.error(`ERROR: ${error.message}`);
    process.exit(1);
```

**Exact new block:**

```js
  } catch (error) {
    console.error(`ERROR: ${error.message}`);
    throw error;
```
