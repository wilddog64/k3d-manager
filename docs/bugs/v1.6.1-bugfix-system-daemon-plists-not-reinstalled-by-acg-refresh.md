# Bugfix: v1.6.1 — System daemon plists not reinstalled by acg-refresh

**Branch:** `k3d-manager-v1.6.1`
**Files:** `bin/acg-refresh`

---

## Problem

`acg-down` removes three system daemon plists from `/Library/LaunchDaemons/`:
- `com.k3d-manager.argocd-browser-https` (socat TLS proxy → port 443)
- `com.k3d-manager.keycloak-browser-http` (kubectl port-forward → Istio IngressGateway port 80)
- `com.k3d-manager.frontend-browser-http` (kubectl port-forward → frontend port 127.0.0.2:80)

`acg-up` Step 10e reinstalls them. If `acg-up` is interrupted before Step 10e, the plists are
gone. `acg-refresh` calls `_launchd_ensure` for each, but `_launchd_ensure` silently returns
when the plist file is missing (`[[ -f "${plist}" ]] || return 0`). The result: ports 80 and 443
stay closed after any incomplete rebuild, and `prometheus.3ai-talk.org`,
`grafana.3ai-talk.org`, `argocd.3ai-talk.org`, and `keycloak.3ai-talk.org` all return 502.
The user must run manual `sudo cp` + `sudo launchctl bootstrap` commands to recover.

**Root cause:** `_launchd_ensure` only reloads existing plists. No code path in `acg-refresh`
regenerates a missing system daemon plist from its wrapper script.

---

## Reproduction

1. Run `bin/acg-down` (removes all three plists)
2. Interrupt `bin/acg-up` before Step 10e
3. Run `make refresh`
4. `nc -zv localhost 80` → "Connection refused"
5. `prometheus.3ai-talk.org` and `grafana.3ai-talk.org` → 502

### Change 1 — add `_system_daemon_install_if_missing` helper after `_launchd_restart` (~line 104)

```bash
}

```

**Exact new block:**
```bash

# Regenerate a system daemon plist from its wrapper script when the plist is missing.
# Writes plist to a temp path, then sudo-copies it to /Library/LaunchDaemons/.
# No-op if plist already exists or wrapper is absent.
# Usage: _system_daemon_install_if_missing <label> <wrapper> <log> <plist-dst>
_system_daemon_install_if_missing() {
  local label="$1" wrapper="$2" log="$3" plist_dst="$4"
  local plist_tmp="${_ACG_STATE_DIR}/run/${label}.plist"
  [[ -f "${plist_dst}" ]] && return 0
  [[ -f "${wrapper}" ]] || return 0
  _info "[acg-refresh] ${label} plist missing — regenerating from wrapper..."
  mkdir -p "$(dirname "${log}")" "$(dirname "${plist_tmp}")"
  cat > "${plist_tmp}" <<PLIST
<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE plist PUBLIC "-//Apple//DTD PLIST 1.0//EN" "http://www.apple.com/DTDs/PropertyList-1.0.dtd">
<plist version="1.0">
<dict>
  <key>Label</key>
  <string>${label}</string>
  <key>ProgramArguments</key>
  <array>
    <string>/bin/bash</string>
    <string>${wrapper}</string>
  </array>
  <key>RunAtLoad</key>
  <true/>
  <key>KeepAlive</key>
  <true/>
  <key>StandardOutPath</key>
  <string>${log}</string>
  <key>StandardErrorPath</key>
  <string>${log}</string>
</dict>
</plist>
PLIST
  _run_command --prefer-sudo --quiet --soft -- cp "${plist_tmp}" "${plist_dst}" \
    && _info "[acg-refresh] ${label} plist installed" \
    || _warn "[acg-refresh] ${label} plist install failed — sudo may be required"
}

```

### Change 2 — replace the three bare `_launchd_ensure` system daemon calls (~lines 216–232)

**Exact old block:**
```bash
  # ── 5. launchd daemons (system-domain) ───────────────────────────────────────
  _info "[acg-refresh] Checking launchd system daemons..."

  _launchd_ensure \
    "com.k3d-manager.argocd-browser-https" \
    "/Library/LaunchDaemons/com.k3d-manager.argocd-browser-https.plist" \
    system

  _launchd_ensure \
    "com.k3d-manager.keycloak-browser-http" \
    "/Library/LaunchDaemons/com.k3d-manager.keycloak-browser-http.plist" \
    system

  _launchd_ensure \
    "com.k3d-manager.frontend-browser-http" \
    "/Library/LaunchDaemons/com.k3d-manager.frontend-browser-http.plist" \
    system
```

**Exact new block:**
```bash
  # ── 5. launchd daemons (system-domain) ───────────────────────────────────────
  _info "[acg-refresh] Checking launchd system daemons..."

  _system_daemon_install_if_missing \
    "com.k3d-manager.argocd-browser-https" \
    "${_ACG_STATE_DIR}/bin/argocd-browser-https.sh" \
    "${_ACG_STATE_DIR}/logs/argocd-browser-https.log" \
    "/Library/LaunchDaemons/com.k3d-manager.argocd-browser-https.plist"
  _launchd_ensure \
    "com.k3d-manager.argocd-browser-https" \
    "/Library/LaunchDaemons/com.k3d-manager.argocd-browser-https.plist" \
    system

  _system_daemon_install_if_missing \
    "com.k3d-manager.keycloak-browser-http" \
    "${_ACG_STATE_DIR}/bin/keycloak-browser-http.sh" \
    "${_ACG_STATE_DIR}/logs/keycloak-browser-http.log" \
    "/Library/LaunchDaemons/com.k3d-manager.keycloak-browser-http.plist"
  _launchd_ensure \
    "com.k3d-manager.keycloak-browser-http" \
    "/Library/LaunchDaemons/com.k3d-manager.keycloak-browser-http.plist" \
    system

  _frontend_browser_wrapper=""
  if [[ -f "${_ACG_STATE_DIR}/bin/frontend-browser-http.sh" ]]; then
    _frontend_browser_wrapper="${_ACG_STATE_DIR}/bin/frontend-browser-http.sh"
  elif [[ -f "${_ACG_STATE_DIR}/frontend-browser-http.sh" ]]; then
    _frontend_browser_wrapper="${_ACG_STATE_DIR}/frontend-browser-http.sh"
  fi
  if [[ -n "${_frontend_browser_wrapper}" ]]; then
    _system_daemon_install_if_missing \
      "com.k3d-manager.frontend-browser-http" \
      "${_frontend_browser_wrapper}" \
      "${_ACG_STATE_DIR}/logs/frontend-browser-http.log" \
      "/Library/LaunchDaemons/com.k3d-manager.frontend-browser-http.plist"
  fi
  _launchd_ensure \
    "com.k3d-manager.frontend-browser-http" \
    "/Library/LaunchDaemons/com.k3d-manager.frontend-browser-http.plist" \
    system
```
