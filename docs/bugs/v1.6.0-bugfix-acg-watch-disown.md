# Bugfix: v1.6.0 — acg_watch backgrounded without disown blocks webhook stdout pipe

**Branch:** `k3d-manager-v1.6.0`
**Files:** `scripts/lib/providers/k3s-aws.sh`

---

## Problem

When `acg-up` is triggered from `k3dm-webhook`, the webhook's `for line in proc.stdout` loop never gets EOF and blocks indefinitely. The webhook thread stays alive for the full lifetime of the `acg_watch` background process — 7+ hours in the incident of 2026-06-03 (422 minutes until manual kill).

**Root cause:** `acg_watch &` at `scripts/lib/providers/k3s-aws.sh:126` is backgrounded without `disown`. `acg_watch` is an infinite `while true` loop with a 12,600-second sleep. Because it is not disowned, it inherits the webhook subprocess's stdout pipe write end. Python's `for line in proc.stdout` blocks until EOF; EOF is only delivered when all processes holding the write end close it. `acg_watch` keeps that fd open, so the loop never exits.

---

## Reproduction

```
# Trigger acg-up from the Slack webhook
# → make up completes after ~40 min
# → webhook _run_cluster thread never exits
# → Slack sends progress updates every 2 min forever
# → Eventually: sandbox expires, acg_watch wakes at 3.5h, extend fails,
#   prints extend failure messages to the inherited pipe (visible in Slack output)
# → Process must be killed manually
```

Expected: webhook thread exits within a few seconds of `make up` completing.
Actual: webhook thread blocks for 7+ hours until `acg_watch` is externally killed.

---

## Fix

### Change 1 — `scripts/lib/providers/k3s-aws.sh`: add `disown` after `acg_watch &`

**Exact old block (lines 125–131):**

```bash
  if [[ ! -f "${_ACG_WATCH_PID_FILE}" ]]; then
    acg_watch &
    local _watcher_pid=$!
    mkdir -p "$(dirname "${_ACG_WATCH_PID_FILE}")"
    printf '%s\n' "${_watcher_pid}" > "${_ACG_WATCH_PID_FILE}"
    _info "[k3s-aws] Watcher PID: ${_watcher_pid} (stored in ${_ACG_WATCH_PID_FILE})"
  fi
```

**Exact new block:**

```bash
  if [[ ! -f "${_ACG_WATCH_PID_FILE}" ]]; then
    acg_watch &
    local _watcher_pid=$!
    disown "${_watcher_pid}"
    mkdir -p "$(dirname "${_ACG_WATCH_PID_FILE}")"
    printf '%s\n' "${_watcher_pid}" > "${_ACG_WATCH_PID_FILE}"
    _info "[k3s-aws] Watcher PID: ${_watcher_pid} (stored in ${_ACG_WATCH_PID_FILE})"
  fi
```
