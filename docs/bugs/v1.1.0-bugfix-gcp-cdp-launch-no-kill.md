# Bugfix: v1.1.0 — gcp_get_credentials kills user's Chrome session

**Branch:** `k3d-manager-v1.1.0`
**Files:** `scripts/plugins/gcp.sh`

---

## Before You Start

1. `git pull origin k3d-manager-v1.1.0`
2. Read `scripts/plugins/gcp.sh` lines 88–110 (the current CDP check block)

---

## Problem

The current CDP check block unconditionally runs `pkill -x "Google Chrome"` before
relaunching Chrome. This kills every open Chrome window and tab the user has, even
when Chrome is simply missing the `--remote-debugging-port=9222` flag. This is
destructive and wrong.

The correct behavior:

| Chrome state | Correct action |
|---|---|
| Not running at all | Launch Chrome with `--remote-debugging-port=9222` |
| Running **without** CDP on 9222 | Print a clear error; `return 1` — do NOT kill |
| Running **with** CDP on 9222 | Already handled by outer `if` guard — skip block |

---

## Fix

### Change — `gcp_get_credentials` CDP block (lines 88–110)

**Exact old block:**

```bash
  if ! curl -sf http://localhost:9222/json >/dev/null 2>&1; then
    _info "[gcp] Chrome not running on CDP port 9222 — launching Chrome with your default profile..."
    if [[ "$(uname)" == "Darwin" ]]; then
      pkill -x "Google Chrome" 2>/dev/null || true
      sleep 1
      open -a "Google Chrome" --args --remote-debugging-port=9222
    else
      local _chrome_bin
      _chrome_bin=$(command -v google-chrome 2>/dev/null || command -v google-chrome-stable 2>/dev/null || command -v chromium-browser 2>/dev/null || command -v chromium 2>/dev/null || true)
      if [[ -z "${_chrome_bin}" ]]; then
        _err "[gcp] Chrome/Chromium not found — install google-chrome or chromium"
      fi
      "${_chrome_bin}" --remote-debugging-port=9222 &
    fi
    local _cdp_deadline=$(( $(date +%s) + 15 ))
    until curl -sf http://localhost:9222/json >/dev/null 2>&1; do
      if (( $(date +%s) >= _cdp_deadline )); then
        _err "[gcp] Timed out waiting for Chrome CDP on port 9222"
      fi
      sleep 1
    done
    _info "[gcp] Chrome CDP ready on port 9222"
  fi
```

**Exact new block:**

```bash
  if ! curl -sf http://localhost:9222/json >/dev/null 2>&1; then
    if [[ "$(uname)" == "Darwin" ]]; then
      if pgrep -x "Google Chrome" >/dev/null 2>&1; then
        _err "[gcp] Chrome is running but CDP is not enabled. Quit Chrome and rerun: open -a \"Google Chrome\" --args --remote-debugging-port=9222"
        return 1
      fi
      _info "[gcp] Chrome not running — launching with CDP port 9222 and your default profile..."
      open -a "Google Chrome" --args --remote-debugging-port=9222
    else
      local _chrome_bin
      _chrome_bin=$(command -v google-chrome 2>/dev/null || command -v google-chrome-stable 2>/dev/null || command -v chromium-browser 2>/dev/null || command -v chromium 2>/dev/null || true)
      if [[ -z "${_chrome_bin}" ]]; then
        _err "[gcp] Chrome/Chromium not found — install google-chrome or chromium"
        return 1
      fi
      if pgrep -x "google-chrome" >/dev/null 2>&1 || pgrep -x "google-chrome-stable" >/dev/null 2>&1 || pgrep -x "chromium-browser" >/dev/null 2>&1 || pgrep -x "chromium" >/dev/null 2>&1; then
        _err "[gcp] Chrome is running but CDP is not enabled. Quit Chrome and rerun: ${_chrome_bin} --remote-debugging-port=9222"
        return 1
      fi
      _info "[gcp] Chrome not running — launching with CDP port 9222..."
      "${_chrome_bin}" --remote-debugging-port=9222 &
    fi
    local _cdp_deadline=$(( $(date +%s) + 15 ))
    until curl -sf http://localhost:9222/json >/dev/null 2>&1; do
      if (( $(date +%s) >= _cdp_deadline )); then
        _err "[gcp] Timed out waiting for Chrome CDP on port 9222"
        return 1
      fi
      sleep 1
    done
    _info "[gcp] Chrome CDP ready on port 9222"
  fi
```
