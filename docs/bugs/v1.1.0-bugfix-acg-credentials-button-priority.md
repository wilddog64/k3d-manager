## Problem

When `make up CLUSTER_PROVIDER=k3s-gcp` runs with a GCP sandbox already active,
the Pluralsight cloud-sandboxes page shows the GCP card with **"Open Sandbox"** and
the AWS card with **"Start Sandbox"** (disabled). The script checks "Start Sandbox"
first in its `if/else if` chain, finds the AWS disabled button, attempts to click it,
and times out after 30 seconds.

**Root cause:** The `if/else if` priority at lines 340–360 of
`scripts/playwright/acg_credentials.js` checks "Start Sandbox" before "Open Sandbox".
"Open Sandbox" is the correct entry point when the target provider's sandbox is already
running, but the code never reaches it because "Start Sandbox" is visible first (on
the wrong provider's card).

---

## Reproduction

```bash
# GCP sandbox already running in Pluralsight
make up CLUSTER_PROVIDER=k3s-gcp
# → INFO: Clicking Start Sandbox...
# → ERROR: locator.click: Timeout 30000ms exceeded
# → element is not enabled  (AWS "Start Sandbox" is disabled when GCP is active)
# → ERROR: [gcp] Failed to extract credentials via Playwright
```

---

## Fix

### Change 1 — `scripts/playwright/acg_credentials.js`: swap Open before Start in priority

**Exact old block (lines 340–360):**

```js
      if (await startButton.isVisible({ timeout: 5000 }).catch(() => false)) {
        console.error('INFO: Clicking Start Sandbox...');
        await startButton.click();
        await _waitForCredentials();
      } else if (await openButton.isVisible({ timeout: 5000 }).catch(() => false)) {
        console.error('INFO: Clicking Open Sandbox...');
        await openButton.click();
        await page.waitForTimeout(3000);

        // After Open, there might be a Start Sandbox button in the slide-over
        const startButton2 = _scope.locator('button:has-text("Start Sandbox")').first();
        if (await startButton2.isVisible({ timeout: 5000 }).catch(() => false)) {
          console.error('INFO: Clicking Start Sandbox (Step 2)...');
          await startButton2.click();
        }
        await _waitForCredentials();
      } else if (await resumeButton.isVisible({ timeout: 5000 }).catch(() => false)) {
        console.error('INFO: Clicking Resume Sandbox...');
        await resumeButton.click();
        await _waitForCredentials();
      }
```

**Exact new block:**

```js
      if (await openButton.isVisible({ timeout: 5000 }).catch(() => false)) {

        const startButton2 = _scope.locator('button:has-text("Start Sandbox")').first();

          console.error('INFO: Clicking Start Sandbox (Step 2)...');
          await startButton2.click();
        }
        await _waitForCredentials();
      } else if (await startButton.isVisible({ timeout: 5000 }).catch(() => false)) {
        console.error('INFO: Clicking Start Sandbox...');

      } else if (await resumeButton.isVisible({ timeout: 5000 }).catch(() => false)) {
        console.error('INFO: Clicking Resume Sandbox...');
        await resumeButton.click();
        await _waitForCredentials();
      }
```
