## Problem

The ArgoCD Image Updater controller is installed only by a manual
`kubectl apply -k scripts/etc/argocd/image-updater/` (documented in
`docs/plans/v1.4.9-argocd-image-updater.md`). No `argocd.sh` function performs
this install, so it is **not part of cluster bootstrap**. When the hub k3d
cluster is rebuilt, the Image Updater controller is gone and no app images are
auto-updated — even the Phase 2 annotations on basket/order/product-catalog do
nothing because the controller that consumes them is absent.

Verified live on `k3d-k3d-cluster` (2026-06-28): no `deploy/sa/cm` named
`argocd-image-updater` exists in the `cicd` namespace, and `image-updater` is
referenced only under `scripts/etc/argocd/` — never in any `scripts/plugins/*.sh`.

**Root cause:** `_argocd_configure_post_deploy` deploys the AppProject and
ApplicationSets in its `enable_bootstrap` block but never installs the Image
Updater controller, so the controller install is not reproducible from
`deploy_argocd_bootstrap` and does not survive a cluster rebuild.

## Reproduction

```bash
# After a fresh hub bootstrap:
kubectl --context k3d-k3d-cluster -n cicd get deploy argocd-image-updater
# Expected: Deployment present, READY 1/1
# Actual:   Error from server (NotFound): deployments.apps "argocd-image-updater" not found
```

## Fix

The existing `scripts/etc/argocd/image-updater/` kustomization is known-good
(it ran product-catalog Image Updater in production). The fix only **automates**
the existing install by adding a private deploy function and calling it from the
bootstrap path — it does NOT change the kustomization or its cross-cluster
pull-secret config.

### Change 1 — `scripts/plugins/argocd.sh`: add `_argocd_deploy_image_updater`

Insert the new function immediately **before** `_argocd_deploy_applicationsets()`.

**Exact old block (anchor — line ~1073):**

```bash
function _argocd_deploy_applicationsets() {
   _info "[argocd] Deploying sample ApplicationSets"
```

**Exact new block:**

```bash
function _argocd_deploy_image_updater() {
   if [[ "${ARGOCD_SKIP_IMAGE_UPDATER:-0}" == "1" ]]; then
      _info "[argocd] Skipping ArgoCD Image Updater install (ARGOCD_SKIP_IMAGE_UPDATER=1)"
      return 0
   fi

   local updater_dir="$ARGOCD_CONFIG_DIR/image-updater"
   if [[ ! -d "$updater_dir" ]]; then
      _warn "[argocd] Image Updater config dir not found: $updater_dir"
      return 0
   fi

   _info "[argocd] Installing ArgoCD Image Updater (v0.15.0)"
   if ! _kubectl apply -k "$updater_dir" >/dev/null 2>&1; then
      _warn "[argocd] Image Updater install failed (apply -k); continuing"
      return 0
   fi

   _kubectl -n "$ARGOCD_NAMESPACE" rollout restart deploy/argocd-image-updater >/dev/null 2>&1 || true
   if ! _kubectl --no-exit -n "$ARGOCD_NAMESPACE" rollout status deploy/argocd-image-updater --timeout=120s; then
      _warn "[argocd] Image Updater not Ready within timeout; check: kubectl -n $ARGOCD_NAMESPACE get deploy argocd-image-updater"
   fi
   _info "[argocd] ArgoCD Image Updater install complete"
}

function _argocd_deploy_applicationsets() {
   _info "[argocd] Deploying sample ApplicationSets"
```

### Change 2 — `scripts/plugins/argocd.sh`: call it from bootstrap

In `_argocd_configure_post_deploy`, invoke the new function at the start of the
`enable_bootstrap` block so the controller exists before the ApplicationSets
(which carry the Image Updater annotations) are created.

**Exact old block (lines ~550–554):**

```bash
   if (( enable_bootstrap )); then
      _info "[argocd] Deploying GitOps bootstrap resources"
      if (( ! skip_appproject )); then
         _argocd_deploy_appproject
      fi
```

**Exact new block:**

```bash
   if (( enable_bootstrap )); then
      _info "[argocd] Deploying GitOps bootstrap resources"
      _argocd_deploy_image_updater
      if (( ! skip_appproject )); then
         _argocd_deploy_appproject
      fi
```

### Change 3 — new test `scripts/tests/plugins/argocd_image_updater_install.bats`

Create the file with exactly these contents:

```bash
#!/usr/bin/env bats

SRC="${BATS_TEST_DIRNAME}/../../plugins/argocd.sh"

@test "image-updater install: deploy function defined" {
  run grep -F -- 'function _argocd_deploy_image_updater()' "${SRC}"
  [ "${status}" -eq 0 ]
}

@test "image-updater install: targets image-updater config dir" {
  run grep -F -- 'updater_dir="$ARGOCD_CONFIG_DIR/image-updater"' "${SRC}"
  [ "${status}" -eq 0 ]
}

@test "image-updater install: applies kustomize dir" {
  run grep -F -- 'apply -k "$updater_dir"' "${SRC}"
  [ "${status}" -eq 0 ]
}

@test "image-updater install: honors ARGOCD_SKIP_IMAGE_UPDATER" {
  run grep -F -- 'ARGOCD_SKIP_IMAGE_UPDATER' "${SRC}"
  [ "${status}" -eq 0 ]
}

@test "image-updater install: invoked from post-deploy bootstrap" {
  run grep -cF -- '_argocd_deploy_image_updater' "${SRC}"
  [ "${status}" -eq 0 ]
  [ "${output}" -ge 2 ]
}
```
