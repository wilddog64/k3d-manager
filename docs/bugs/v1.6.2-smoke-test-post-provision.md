# Feature: v1.6.2 — Post-provision service smoke test

**Branch:** `k3d-manager-v1.6.2`
**Files:** `bin/k3dm-webhook`

---

## Problem

`_run_post_provision_check` only checks ArgoCD app `Health` status from the Hub cluster.
It does not verify that services actually respond to HTTP — ArgoCD can show all apps
Healthy while a service is crashing, a port-forward is broken, or product images are
missing from the catalog. These failures are discovered manually.

---

## Fix

Add a `_smoke_test_services(job_id, provider)` function and call it from
`_run_post_provision_check` after the existing ArgoCD degraded-app check.

The smoke test hits five endpoints via `urllib.request`, retrying up to 3 times with
a 10-second gap between attempts to tolerate timing windows right after provisioning.
Results are appended to the post-provision Slack message.

### Change 1 — `bin/k3dm-webhook`: add `_smoke_test_services` function

Insert the following function **before** `_run_post_provision_check` (i.e. before line 712).

**Exact new block to insert:**

```python
_SMOKE_ENDPOINTS = [
    ("ArgoCD",     "https://argocd.shopping-cart.local/healthz",      [200, 307]),
    ("Frontend",   "http://frontend.shopping-cart.local/",             [200]),
    ("Prometheus", "http://localhost:19090/-/ready",                   [200]),
    ("Grafana",    "https://grafana.3ai-talk.org/api/health",          [200]),
]
_SMOKE_RETRIES = 3
_SMOKE_RETRY_SLEEP = 10


def _smoke_test_services():
    """HTTP smoke test for each service. Returns list of (name, ok, detail) tuples."""
    import ssl
    import time
    ctx = ssl.create_default_context()
    ctx.check_hostname = False
    ctx.verify_mode = ssl.CERT_NONE

    results = []
    for name, url, ok_codes in _SMOKE_ENDPOINTS:
        last_err = ""
        passed = False
        for attempt in range(_SMOKE_RETRIES):
            try:
                req = urllib.request.Request(url, headers={"User-Agent": "k3dm-smoketest/1"})
                with urllib.request.urlopen(req, timeout=8, context=ctx) as resp:
                    code = resp.status
                    if code in ok_codes:
                        passed = True
                        last_err = f"HTTP {code}"
                        break
                    last_err = f"HTTP {code}"
            except Exception as exc:
                last_err = str(exc)
            if attempt < _SMOKE_RETRIES - 1:
                time.sleep(_SMOKE_RETRY_SLEEP)
        results.append((name, passed, last_err))

    # Product catalog: verify at least one product has a non-empty imageUrl
    try:
        req = urllib.request.Request(
            "http://frontend.shopping-cart.local/api/products",
            headers={"User-Agent": "k3dm-smoketest/1"},
        )
        with urllib.request.urlopen(req, timeout=8) as resp:
            products = json.loads(resp.read())
            if isinstance(products, list) and products:
                with_images = sum(1 for p in products if p.get("imageUrl") or p.get("image_url"))
                if with_images > 0:
                    results.append(("Product images", True, f"{with_images}/{len(products)} have imageUrl"))
                else:
                    results.append(("Product images", False, f"0/{len(products)} have imageUrl"))
            else:
                results.append(("Product images", False, "empty or non-list response"))
    except Exception as exc:
        results.append(("Product images", False, str(exc)))

    return results
```

### Change 3 — `bin/k3dm-webhook`: Gemini triage for failed smoke checks

Add a smoke-test triage block **after** the existing AI-triage loop for degraded ArgoCD
apps (i.e. after the closing `for app_name in degraded[:5]:` block, still inside
`_run_post_provision_check`).

**Exact new block to append at the end of `_run_post_provision_check`:**

```python
    # Gemini triage for failed smoke checks
    failed_smoke = [(name, detail) for name, ok, detail in smoke if not ok]
    for svc_name, err_detail in failed_smoke[:3]:  # cap at 3
        diag_parts = [f"Service `{svc_name}` failed smoke test: {err_detail}"]

        # Gather relevant pod state from ubuntu-k3s
        ns_map = {
            "Frontend":       ("shopping-cart-apps", "app=frontend"),
            "Product images": ("shopping-cart-apps", "app=product-catalog"),
            "Prometheus":     ("monitoring",          "app.kubernetes.io/name=prometheus"),
            "Grafana":        ("monitoring",          "app.kubernetes.io/name=grafana"),
            "ArgoCD":         ("cicd",                "app.kubernetes.io/name=argocd-server"),
        }
        ns, label_sel = ns_map.get(svc_name, ("shopping-cart-apps", ""))
        if label_sel:
            pod_state = subprocess.run(
                ["kubectl", "get", "pods", "-n", ns, "-l", label_sel,
                 "--context", "ubuntu-k3s", "--no-headers",
                 "--request-timeout=8s"],
                capture_output=True, text=True, timeout=15,
            )
            if pod_state.returncode == 0 and pod_state.stdout.strip():
                diag_parts.append(f"Pod state ({ns}):\n{pod_state.stdout.strip()[:800]}")

            pod_logs = subprocess.run(
                ["kubectl", "logs", "-n", ns, "-l", label_sel,
                 "--context", "ubuntu-k3s", "--tail=30",
                 "--request-timeout=8s"],
                capture_output=True, text=True, timeout=15,
            )
            if pod_logs.returncode == 0 and pod_logs.stdout.strip():
                diag_parts.append(f"Recent logs ({ns}):\n{pod_logs.stdout.strip()[-1000:]}")

        prompt = (
            f"Service `{svc_name}` failed an HTTP smoke test 90 seconds after a fresh "
            f"Kubernetes cluster provision. Error: {err_detail}\n\n"
            "Diagnostics:\n" + "\n\n".join(diag_parts) + "\n\n"
            "Is this a TRANSIENT issue (pod still starting, port-forward not ready — "
            "will resolve in a few minutes) or a REAL FAILURE (crash loop, missing secret, "
            "misconfiguration — needs human action)?\n"
            "Reply with TRANSIENT or REAL FAILURE, then one sentence with the most likely "
            "cause and what to check or fix. Be brief."
        )
        analysis = _call_gemini(prompt)
        _notify_job(job_id, f"🔍 *{svc_name} smoke failure* — {analysis}")
```
