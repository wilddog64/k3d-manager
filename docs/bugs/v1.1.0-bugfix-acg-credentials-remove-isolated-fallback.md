## Problem

When Chrome is not running with `--remote-debugging-port=9222`, the script falls
back to `chromium.launchPersistentContext(AUTH_DIR)` ‚Äî an isolated browser profile
with no Pluralsight session cookies. This opens a new separate Chrome window and
requires the user to log in again, even though they are already logged into
Pluralsight in their real Chrome.

This is wrong behavior. The user should never be asked to log in again when they
already have a valid Pluralsight session. The correct behavior is:

- If Chrome is running on port 9222 ‚Üí connect via CDP and use the existing session
- If Chrome is not running on port 9222 ‚Üí exit with a clear error and tell the user
  exactly how to fix it

The `launchPersistentContext` isolated-profile fallback must be removed entirely.
CDP is required.

---

## Fix

### Change 1 ‚Äî inner catch block (lines 166‚Äì170)

Replace the entire catch block with an actionable error throw.

**Exact old block:**

```js
    } catch {
      console.error('INFO: Chrome not running on CDP port 9222 ‚Äî falling back to isolated Playwright profile.');
      console.error('INFO: A new Chrome window will open. Log in to Pluralsight there, OR restart Chrome with --remote-debugging-port=9222 to reuse your existing session automatically.');
      _cdpBrowser = null;
    }
```

**Exact new block:**

```js
    } catch {
      console.error('ERROR: Chrome is not running with --remote-debugging-port=9222.');
      console.error('ERROR: Start Chrome with CDP enabled, then rerun:');
      console.error('ERROR:   open -a "Google Chrome" --args --remote-debugging-port=9222');
      throw new Error('Chrome not running with --remote-debugging-port=9222 ‚Äî cannot access Pluralsight session');
    }
```

### Change 2 ‚Äî launchPersistentContext fallback (lines 171‚Äì177)

Replace the isolated-profile fallback with an error for the empty-contexts case.

**Exact old block:**

```js
    if (!browserContext) {
      browserContext = await chromium.launchPersistentContext(AUTH_DIR, {
        headless: false,
        channel: 'chrome',
        args: ['--password-store=basic'],
      });
    }
```

**Exact new block:**

```js
    if (!browserContext) {
      throw new Error('CDP connected but no browser contexts found ‚Äî restart Chrome with --remote-debugging-port=9222 and try again');
    }
```
