# Bug: `ubuntu-k3s-platform` app stuck in InvalidSpecError ‚Äî platform project missing `cicd/ubuntu-k3s` destination

**Branch:** `k3d-manager-v1.5.0`
**File:** `scripts/etc/argocd/projects/platform.yaml.tmpl`
**Severity:** P2 ‚Äî `ubuntu-k3s-platform` ArgoCD app is permanently `Unknown/Unknown`

---

## Symptom

After `make up`, the `ubuntu-k3s-platform` app (generated by the `platform-helm` ApplicationSet)
shows:

```
cicd  ubuntu-k3s-platform  Unknown  Unknown
```

`argocd app get ubuntu-k3s-platform` reports:

```
CONDITION         MESSAGE
InvalidSpecError  application destination server 'https://35.93.5.150:6443' and namespace 'cicd'
                  do not match any of the allowed destinations in project 'platform'
```

---

## Root Cause

The `platform-helm` ApplicationSet (updated in commit `0e4268a5`) now selects clusters with
`environment: [dev, infra, prod]`. The `ubuntu-k3s` cluster has `environment: dev` and
`argocd-chart-version: 7.8.1` labels on its ArgoCD cluster secret, so the ApplicationSet
generates an app (`ubuntu-k3s-platform`) that deploys ArgoCD to the `cicd` namespace on
the ubuntu-k3s cluster.

However, `platform.yaml.tmpl` only allows the `cicd` namespace for `https://kubernetes.default.svc`
(the hub). It has no entry allowing `cicd` for the ubuntu-k3s cluster:

```yaml
destinations:
  - namespace: cicd
    server: https://kubernetes.default.svc   # hub only
  # ... ubuntu-k3s entries only cover shopping-cart-apps, shopping-cart-payment, shopping-cart-data
```

---

## Fix

**`scripts/etc/argocd/projects/platform.yaml.tmpl` ‚Äî add one destination entry after line 31:**

```yaml
# current line 29‚Äì31:
    - namespace: shopping-cart-apps
      name: ubuntu-k3s
    - namespace: shopping-cart-payment
      name: ubuntu-k3s
    - namespace: shopping-cart-data
      name: ubuntu-k3s
```

Replace with:

```yaml
    - namespace: shopping-cart-apps
      name: ubuntu-k3s
    - namespace: shopping-cart-payment
      name: ubuntu-k3s
    - namespace: shopping-cart-data
      name: ubuntu-k3s
    - namespace: cicd
      name: ubuntu-k3s
```
