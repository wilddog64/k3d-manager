# Bug: acg-up Docker preflight fails because OrbStack VM is stopped — open -a is a no-op

**Date:** 2026-06-11
**Branch:** k3d-manager-v1.6.5
**File:** `bin/acg-up`

---

## Symptom

`make up` fails at the Docker preflight step even with the fix from `4adf5a1a`:

```
INFO: [acg-up] Docker daemon not responding — dismissing any OrbStack update dialogs...
INFO: [acg-up] Docker still unavailable — launching OrbStack...
ERROR: [acg-up] Docker daemon unavailable after recovery — ...
make: *** [up] Error 1
```

OrbStack shows "Starting containers" in its UI. Restarting OrbStack from the menu bar
does not resolve it.

---

## Root Cause

Diagnosed via:

```
$ ls -la /var/run/docker.sock
/var/run/docker.sock -> /Users/cliang/.orbstack/run/docker.sock  # symlink exists

$ ls ~/.orbstack/run/docker.sock
No such file or directory   # socket target MISSING — VM not running

$ orb status
Stopped
```

**The OrbStack Linux VM is stopped.** The macOS OrbStack app is running (visible in UI),
but its Linux VM is not. `~/.orbstack/run/docker.sock` is only created by the VM, so
`docker info` always fails until the VM starts.

The `osascript` dialog-dismiss and `open -a OrbStack` fallback added in `4adf5a1a` were
based on the wrong hypothesis (update dialogs blocking the daemon). Neither addresses a
stopped VM. `open -a OrbStack` on an already-running macOS app is a no-op — it does not
start the VM. The correct command is `orbctl start`.

---

## Fix

Replace the entire preflight block (lines 202–247 as of `4adf5a1a`) with a clean
`orbctl`-first recovery that removes the `osascript` and `open -a` fallback entirely.

### Change 1 — `bin/acg-up`: replace Docker preflight block

**Exact old block (lines 202–247):**

```bash
if ! docker info >/dev/null 2>&1; then
  _info "[acg-up] Docker daemon not responding — dismissing any OrbStack update dialogs..."
  osascript 2>/dev/null <<'APPLESCRIPT' || true
tell application "System Events"
  if exists process "OrbStack" then
    tell process "OrbStack"
      repeat with w in windows
        if exists button "Remind Me Later" of w then
          click button "Remind Me Later" of w
        end if
        if exists button "Cancel Update" of w then
          click button "Cancel Update" of w
        end if
      end repeat
    end tell
  end if
end tell
APPLESCRIPT
  _docker_wait=0
  until docker info >/dev/null 2>&1 || [[ "${_docker_wait}" -ge 30 ]]; do
    sleep 2
    ((_docker_wait++)) || true
  done
  if ! docker info >/dev/null 2>&1; then
    _docker_app=""
    if [[ -d "/Applications/OrbStack.app" ]]; then
      _docker_app="OrbStack"
    elif [[ -d "/Applications/Docker.app" ]]; then
      _docker_app="Docker"
    fi
    if [[ -n "${_docker_app}" ]]; then
      _info "[acg-up] Docker still unavailable — launching ${_docker_app}..."
      open -a "${_docker_app}"
      _docker_wait=0
      until docker info >/dev/null 2>&1 || [[ "${_docker_wait}" -ge 30 ]]; do
        sleep 2
        ((_docker_wait++)) || true
      done
    fi
  fi
  if ! docker info >/dev/null 2>&1; then
    _err "[acg-up] Docker daemon unavailable after recovery — check OrbStack/Docker Desktop status and dismiss any update dialogs, then re-run"
    exit 1
  fi
  _info "[acg-up] Docker daemon is up."
fi
```

**Exact new block:**

```bash
if ! docker info >/dev/null 2>&1; then
  _info "[acg-up] Docker daemon not responding — checking OrbStack VM state..."
  if command -v orbctl >/dev/null 2>&1; then
    _orb_status=$(orbctl status 2>/dev/null || true)
    if [[ "${_orb_status}" == "Stopped" ]]; then
      _info "[acg-up] OrbStack VM is stopped — starting it now..."
      orbctl start 2>/dev/null || true
    fi
  elif [[ -d "/Applications/OrbStack.app" ]]; then
    _info "[acg-up] orbctl not found — opening OrbStack.app..."
    open -a "OrbStack"
  elif [[ -d "/Applications/Docker.app" ]]; then
    _info "[acg-up] orbctl not found — opening Docker.app..."
    open -a "Docker"
  fi
  _orbstack_sock="${HOME}/.orbstack/run/docker.sock"
  _docker_wait=0
  until (docker info >/dev/null 2>&1 && [[ -S "${_orbstack_sock}" ]]) || \
        [[ "${_docker_wait}" -ge 60 ]]; do
    sleep 2
    ((_docker_wait++)) || true
  done
  if ! docker info >/dev/null 2>&1; then
    _err "[acg-up] Docker daemon unavailable after recovery — run 'orbctl start' and retry"
    exit 1
  fi
  _info "[acg-up] Docker daemon is up."
fi
```
