# Bugfix: v1.6.3 — Gemini CLI warnings leaking into Slack failure analysis

**Branch:** `k3d-manager-v1.6.3`
**Files:** `bin/k3dm-webhook`

---

## Problem

Slack failure analysis messages include Gemini CLI warning banners:

```
Warning: You are running Gemini CLI in the root directory. Your entire folder
structure will be used for context. It is strongly recommended to run in a
project-specific directory.
Warning: 256-color support not detected. Using a terminal with at least
256-color support is recommended for a better visual experience.
```

These are Gemini CLI startup banners that appear before the actual analysis
output. They pollute every failure analysis posted to Slack.

**Root cause:** `_call_gemini` in `bin/k3dm-webhook` does not strip `Warning:`
banner lines from the captured output before returning.

---

## Reproduction

Trigger any job failure. Gemini failure analysis in Slack includes the two
warning lines above the `APP_FAILURE` / `CLUSTER_FAILURE` verdict.

---

## Fix

### Change 1 — `bin/k3dm-webhook`: strip `Warning:` banner lines after output capture

Add a `re.sub` pass to remove lines that start with `Warning:` (Gemini CLI
startup banners).

**Exact old block (lines 789–794):**

```python
        raw = Path(tmp_path).read_text(errors="replace").strip()
        Path(tmp_path).unlink(missing_ok=True)
        raw = raw or "gemini returned no output"
        cleaned = re.sub(r'(?:^|\n)\s*\w+\([^)]{0,500}\)\s*', ' ', raw, flags=re.MULTILINE)
        cleaned = re.sub(r'<ctrl[^>]*>', '', cleaned).strip()
        return cleaned or raw
```

**Exact new block:**

```python
        raw = Path(tmp_path).read_text(errors="replace").strip()
        Path(tmp_path).unlink(missing_ok=True)
        raw = raw or "gemini returned no output"
        # Strip Gemini CLI startup banners (Warning: lines)
        raw = re.sub(r'(?m)^Warning:.*\n?', '', raw).strip()
        raw = raw or "gemini returned no output"
        cleaned = re.sub(r'(?:^|\n)\s*\w+\([^)]{0,500}\)\s*', ' ', raw, flags=re.MULTILINE)
        cleaned = re.sub(r'<ctrl[^>]*>', '', cleaned).strip()
        return cleaned or raw
```
