## Problem

After navigating to Pluralsight, a Cloudflare bot-check page intercepts the request:

```
Checking your browser before accessing the application.
This process is automatic. Your browser will redirect to your requested content shortly.
Please allow up to 5 seconds…
```

The script does not detect this page. The 30s `waitForFunction` sandbox-button wait
runs against the Cloudflare page — which has no Start/Open/Resume buttons or
credential inputs — and times out. Execution proceeds with the wrong page, and
`_extractGcpCredentials` times out waiting for `text=Username`:

```
INFO: Waiting for page content to load...
WARN: Timed out waiting for sandbox buttons or credentials — proceeding anyway
INFO: Selecting GCP Sandbox buttons via data-heap-id
INFO: Extracting credentials (provider: gcp)...
ERROR: page.waitForSelector: Timeout 15000ms exceeded.
Call log:
  - waiting for locator('text=Username') to be visible
```

---

## Fix

Insert a Cloudflare check immediately after navigation completes (before the
`waitForFunction` aria-busy check). Detect `text=Checking your browser before accessing`
and wait for it to disappear (Cloudflare auto-redirects within ~5s). Then wait for
`domcontentloaded` to ensure the real page has settled.

**Exact old block (lines 224–229 of `acg_credentials.js`):**

```js
    // Give it time to render SPA content after navigation changes
    console.error('INFO: Waiting for page content to load...');
    await page.waitForFunction(
      () => !document.querySelector('[aria-busy="true"]'),
      { timeout: 30000 }
    ).catch(() => console.error('WARN: Skeleton loaders did not clear within 30s — proceeding anyway'));
```

**Exact new block:**

```js
    // Handle Cloudflare bot-check page — auto-redirects within ~5s; wait for it to clear
    const _cfText = page.locator('text=Checking your browser before accessing');
    const _isCfPage = await _cfText.isVisible({ timeout: 2000 }).catch(() => false);
    if (_isCfPage) {
      console.error('INFO: Cloudflare bot check detected — waiting for redirect (up to 15s)...');
      await _cfText.waitFor({ state: 'hidden', timeout: 15000 })
        .catch(() => console.error('WARN: Cloudflare check did not clear — proceeding anyway'));
      await page.waitForLoadState('domcontentloaded', { timeout: 15000 }).catch(() => {});
    }

    // Give it time to render SPA content after navigation changes
    console.error('INFO: Waiting for page content to load...');
    await page.waitForFunction(
      () => !document.querySelector('[aria-busy="true"]'),
      { timeout: 30000 }
    ).catch(() => console.error('WARN: Skeleton loaders did not clear within 30s — proceeding anyway'));
```
