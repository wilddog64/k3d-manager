# Bugfix: v1.4.11 — realm-shopping-cart.json path stale after infra repo reorganization

**Branch:** `k3d-manager-v1.4.11`
**Files:** `bin/acg-up`

---

## Problem

Step 10d/14 fails with `sed: .../identity/keycloak/realm-shopping-cart.json: No such file or directory`.
The realm JSON was relocated to `identity/config/` in shopping-cart-infra but `acg-up` still
references the old `identity/keycloak/` path.

**Root cause:** Path hardcoded in `bin/acg-up` was not updated when the file moved.

---

## Fix

```bash
# Before:
"${REPO_ROOT}/../shopping-carts/shopping-cart-infra/identity/keycloak/realm-shopping-cart.json"

# After:
"${REPO_ROOT}/../shopping-carts/shopping-cart-infra/identity/config/realm-shopping-cart.json"
```

---

| File | Change |
|------|--------|
| `bin/acg-up` | Update realm JSON path from `identity/keycloak/` to `identity/config/` |
