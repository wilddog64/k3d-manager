# Bugfix: v1.6.1 — /acg-status shows stale ArgoCD data after "cluster unreachable"

**Branch:** `k3d-manager-v1.6.1`
**File:** `bin/k3dm-webhook`
**Discovered:** 2026-06-05

---

## Problem

When the ACG cluster (ubuntu-k3s) is unreachable, `/acg-status` in Slack shows:

```
❌ ACG cluster unreachable — sandbox may have expired.

ACG ArgoCD apps:
acg-kube-prometheus-stack   Synced   Healthy
acg-trivy-operator          Synced   Healthy
ubuntu-k3s-platform         Synced   Healthy
```

This is contradictory and misleading: "cluster unreachable" followed immediately by
app data showing the cluster is healthy. The ArgoCD app listing always runs because
it reads from the hub cluster (k3d-k3d-cluster), which is always up. The hub caches
the last-known sync/health state of apps targeting ubuntu-k3s, so it shows stale
data without any indication it is stale.

A second issue: "sandbox may have expired" is the only reason shown, which is wrong
when the cluster is mid-rebuild. The user sees a misleading expiry warning during a
normal `acg-up` run.

---

## Root Cause

In `bin/k3dm-webhook`, the ArgoCD apps block (around line 865) runs unconditionally —
it does not check the `acg_reachable` flag set earlier on line 811. So the cached hub
state is always displayed regardless of ACG cluster reachability.

---

## Fix

Two changes to `bin/k3dm-webhook`.

---

### Change 1 — line ~813: soften the unreachable reason

**Exact old line:**
```python
            _log("\n❌ *ACG cluster unreachable* — sandbox may have expired.")
```

**Exact new line:**
```python
            _log("\n❌ *ACG cluster unreachable* — cluster may be rebuilding or sandbox may have expired.")
```

---

### Change 2 — line ~877: add stale caveat to ArgoCD apps header

**Exact old block:**
```python
            if acg_apps:
                _log("\n*ACG ArgoCD apps:*")
                _log(f"```{chr(10).join(acg_apps)}```")
```

**Exact new block:**
```python
            if acg_apps:
                stale_note = " _(cached — ACG cluster was unreachable)_" if not acg_reachable else ""
                _log(f"\n*ACG ArgoCD apps:*{stale_note}")
                _log(f"```{chr(10).join(acg_apps)}```")
```
