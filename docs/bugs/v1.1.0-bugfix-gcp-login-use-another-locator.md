## Problem

The email input times out at 30 seconds after clicking "Use another account" — the click fires but nothing on the page changes.

**Root cause:** Line 63 uses `div:has-text("Use another account")` which matches any ancestor `<div>` containing that text. Playwright's `.first()` returns the first matching element in DOM order — almost certainly a large container div, not the actual clickable link. The click lands on the wrong element, no transition fires, and the email input never appears.

---

## Reproduction

```
make up CLUSTER_PROVIDER=k3s-gcp
# observe:
# INFO: Clicking "Use another account"
# ERROR: locator.waitFor: Timeout 30000ms exceeded.
# waiting for locator('input[type="email"], input#identifierId').first() to be visible
```

---

## Fix

### Change 1 — `scripts/playwright/gcp_login.js`: use `getByText` for precise click target; add URL diagnostics and settle wait

`getByText('Use another account', { exact: true })` targets the most specific element whose text content exactly matches — the actual link/button, not a container.

**Exact old block (lines 62–71):**

```js
    } else {
      const useAnother = page.locator('li:has-text("Use another account"), div:has-text("Use another account")').first();
      const useAnotherVisible = await useAnother.isVisible({ timeout: 5000 }).catch(() => false);
      if (useAnotherVisible) {
        console.error('INFO: Clicking "Use another account"');
        await useAnother.click();
      }
      // Fill email — SPA transition, no navigation event; wait directly for input
      const emailInput = page.locator('input[type="email"], input#identifierId').first();
      await emailInput.waitFor({ timeout: 30000 });
```

**Exact new block:**

```js
    } else {
      const useAnother = page.getByText('Use another account', { exact: true }).first();
      const useAnotherVisible = await useAnother.isVisible({ timeout: 5000 }).catch(() => false);
      if (useAnotherVisible) {
        console.error(`INFO: URL before click: ${page.url()}`);
        console.error('INFO: Clicking "Use another account"');
        await useAnother.click();
        await page.waitForTimeout(1000);
        console.error(`INFO: URL after click: ${page.url()}`);
      }
      // Fill email — SPA transition; wait directly for input
      const emailInput = page.locator('input[type="email"], input#identifierId').first();
      await emailInput.waitFor({ timeout: 30000 });
```
