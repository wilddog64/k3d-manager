## Problem

`_extractGcpCredentials` extracts credentials via `page.getByLabel('Username')`,
`page.getByLabel('Password')`, and `page.getByLabel('Service Account Credentials')`.
All three return empty strings ã the visible labels on the GCP panel are NOT connected
to the input elements via HTML `for`/`id` or `aria-labelledby`, so Playwright's
`getByLabel()` finds nothing.

**Root cause (confirmed by diagnostic dump):** All three GCP credential fields use
`aria-label="Copyable input"` ã the same as AWS. They appear at positions [2], [3],
[4] in `page.locator('input, textarea').all()` (with the sitewide search bar at [0]
and a hidden input at [1]). Positional extraction on
`page.locator('input[aria-label="Copyable input"]').all()` gives inputs[0]=username,
inputs[1]=password, inputs[2]=service account ã identical to the AWS extractor.

## Fix

### Change 1 ã `scripts/playwright/acg_credentials.js`: replace `getByLabel` block (lines 81ã86) with positional `aria-label="Copyable input"` extraction

**Exact old block (lines 81ã86):**

```js
  // Extract by label ã Playwright resolves the associated input/textarea for each label
  const username = await page.getByLabel('Username').inputValue().catch(() => '');
  const password = await page.getByLabel('Password').inputValue().catch(() => '');
  const serviceAccountJson = await page.getByLabel('Service Account Credentials').inputValue().catch(() => '');

  console.error(`INFO: username="${username.slice(0, 30)}" password="${password ? '[set]' : '[empty]'}" sa_json_len=${serviceAccountJson.length}`);
```

**Exact new block:**

```js
  // GCP fields use aria-label="Copyable input" (same as AWS) ã getByLabel() finds nothing
  // because the visible labels are not HTML-associated. Use positional extraction.
  const inputs = await page.locator('input[aria-label="Copyable input"]').all();
  console.error(`INFO: Found ${inputs.length} copyable inputs`);

  const username = inputs.length >= 1 ? await inputs[0].inputValue().catch(() => '') : '';
  const password = inputs.length >= 2 ? await inputs[1].inputValue().catch(() => '') : '';
  const serviceAccountJson = inputs.length >= 3 ? await inputs[2].inputValue().catch(() => '') : '';

  console.error(`INFO: username="${username.slice(0, 30)}" password="${password ? '[set]' : '[empty]'}" sa_json_len=${serviceAccountJson.length}`);
```
