# Bugfix: v1.1.0 — k3sup fails with SSH connection refused on freshly created GCP VM

**Branch:** `k3d-manager-v1.1.0`
**Files:** `scripts/lib/providers/k3s-gcp.sh`

---

## Problem

`make up CLUSTER_PROVIDER=k3s-gcp` fails immediately after VM creation:

```
Error: unable to connect to 34.134.35.80:22 over ssh: dial tcp 34.134.35.80:22: connect: connection refused
```

`k3sup install` is called the moment the external IP is known, but the GCP VM's SSH
daemon is not yet ready — the instance needs 20–40 seconds to fully boot after
`gcloud compute instances create` returns.

**Root cause:** `_provider_k3s_gcp_deploy_cluster` has no SSH readiness probe between
IP resolution and `k3sup install` (lines 182–186 of `scripts/lib/providers/k3s-gcp.sh`).

---

## Reproduction

```bash
# Start a fresh sandbox, delete any existing instance, then:
make up CLUSTER_PROVIDER=k3s-gcp URL=<sandbox-url>
# → k3sup install
# → Error: unable to connect to <ip>:22 over ssh: connection refused
```

---

## Fix

### Change 1 — `scripts/lib/providers/k3s-gcp.sh`: add SSH probe loop before k3sup

Insert a readiness probe between `_gcp_ssh_config_upsert` and `_ensure_k3sup`.
Poll SSH every 10 seconds for up to 5 minutes (30 retries). Use `nc` (netcat) to
check port 22 reachability — faster than a full SSH handshake and avoids host key issues.

**Exact old block (lines 182–185):**

```bash
  _gcp_ssh_config_upsert "${external_ip}"

  _ensure_k3sup || return 1
  _info "[k3s-gcp] Installing k3s via k3sup..."
```

**Exact new block:**

```bash
  _gcp_ssh_config_upsert "${external_ip}"

  _info "[k3s-gcp] Waiting for SSH on ${external_ip}..."
  local _ssh_retries=30
  local _ssh_i=0
  while (( _ssh_i < _ssh_retries )); do
    if nc -z -w 5 "${external_ip}" 22 2>/dev/null; then
      _info "[k3s-gcp] SSH port open on ${external_ip}"
      break
    fi
    _ssh_i=$(( _ssh_i + 1 ))
    _info "[k3s-gcp] SSH not ready yet (${_ssh_i}/${_ssh_retries}) — retrying in 10s..."
    sleep 10
  done
  if (( _ssh_i >= _ssh_retries )); then
    _err "[k3s-gcp] SSH not available on ${external_ip} after $(( _ssh_retries * 10 ))s"
    return 1
  fi

  _ensure_k3sup || return 1
  _info "[k3s-gcp] Installing k3s via k3sup..."
```
