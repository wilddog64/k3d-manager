## Problem

After password entry + Next, Google's enterprise account flow navigates to a
**"Welcome to your new account"** acceptance screen (requires "I understand" click)
before showing the Allow button.

The existing `I understand` handler runs **before** the polling loop — at that point the
page is still on `v3/signin/challenge/pwd` (the 500ms wait after password Next is too
short for navigation), so the handler finds nothing and skips. By the time the polling
loop runs (60s window), the page has navigated to the welcome screen, but the loop has
no handler for "I understand".

Confirmed from `~/.local/share/k3d-manager/gcp_login_pw.log`:
```
INFO: Filled password
INFO: URL after password Next: https://accounts.google.com/v3/signin/challenge/pwd?...
INFO: URL before Allow: https://accounts.google.com/v3/signin/challenge/pwd?...
INFO: visible button: "Next"
INFO: visible button: "Forgot password?"
INFO: page body on Allow timeout:
Welcome to your new accountWelcome to your new account: cloud_user_p_9e441007@linuxacademygclabs.com...
ERROR: locator.waitFor: Timeout 30000ms exceeded.
```

The body dump shows the welcome page appeared during the 60s loop but went unhandled.

## Fix

### Change 1 — Increase password post-Next wait from 500ms to 2000ms

**Exact old line:**

```js
      await page.waitForTimeout(500);
      console.error(`INFO: URL after password Next: ${page.url()}`);
```

**Exact new lines:**

```js
      await page.waitForTimeout(2000);
      console.error(`INFO: URL after password Next: ${page.url()}`);
```

---

### Change 2 — Add "I understand" handler inside the polling loop

The polling loop currently has three branches per tick:
1. `div[data-identifier]` account rows visible → click
2. `accountchooser` URL → re-login
3. `allowBtn` visible → break

Add a fourth branch between branch 2 and branch 3: check for and click "I understand".

**Exact old block** (the Allow-visible check at the bottom of each loop tick):

```js
      const _allowVisible = await allowBtn.isVisible({ timeout: 500 }).catch(() => false);
      if (_allowVisible) {
        _allowFound = true;
        break;
      }
      await page.waitForTimeout(500);
```

**Exact new block:**

```js
      // "I understand" / new-account welcome screen — appears after password on enterprise accounts
      const _iUnderstand = page.locator('button:has-text("I understand"), input[value="I understand"]').first();
      const _iUnderstandVisible = await _iUnderstand.isVisible({ timeout: 500 }).catch(() => false);
      if (_iUnderstandVisible) {
        console.error('INFO: Allow-loop: Clicking "I understand"');
        await _iUnderstand.click();
        await page.waitForTimeout(1000);
        console.error(`INFO: Allow-loop: URL after "I understand": ${page.url()}`);
        continue;
      }
      const _allowVisible = await allowBtn.isVisible({ timeout: 500 }).catch(() => false);
      if (_allowVisible) {
        _allowFound = true;
        break;
      }
      await page.waitForTimeout(500);
```
