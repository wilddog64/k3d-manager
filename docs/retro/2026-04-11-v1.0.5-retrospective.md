# Retrospective — v1.0.5

**Date:** 2026-04-11
**Milestone:** antigravity decoupling + LDAP Vault KV seeding
**PR:** #62 — merged to main (`2a38bf84`); fix-up #63 — merged to main (`71c88b05`)
**Participants:** Claude, Codex, Copilot

## What Went Well

- Antigravity decoupling was clean — `_acg_extend_playwright` moved to `acg.sh` with correct public wrapper and the launchd-generated wrapper updated in the same commit
- LDAP Vault KV seeding (`bin/acg-up`) shipped without issues; ESO ExternalSecret sync path confirmed
- Copilot found substantive bugs in PR #62 (dispatcher private function, stale docs) and PR #63 (missing --help, private call in k3s-aws.sh) — both rounds caught real problems before they hit production
- All Copilot threads resolved promptly with per-finding replies and GraphQL resolution
- Branch protection enforcement held: `enforce_admins` disabled only for the merge window and restored immediately each time

## What Went Wrong

- **PR #62 merged before Copilot reviewed** — CI passed, Claude checked inline comments (empty), and merged. Copilot review is async and posted ~2 minutes after CI. Required creating PR #63 as a corrective fix-up.
- **`acg_extend_playwright` public wrapper was incomplete** — added to satisfy the dispatcher contract but not audited for the standard public function requirements (--help handler, BATS test). Copilot caught both gaps in the PR #63 review.
- **k3s-aws.sh still called private `_acg_extend_playwright`** — when the public wrapper was added, existing callers outside the dispatcher were not updated. Should have been part of the same commit.
- **Two Copilot review rounds needed** — one finding in PR #62 (launchd wrapper calls private function) led to a fix that introduced three more findings in PR #63 (incomplete wrapper). Better initial audit would have caught these together.

## Decisions Made

- **v1.0.5 tag points to PR #63 merge SHA (`71c88b05`)** — the PR #62 squash merge (`2a38bf84`) was missing the Copilot fixes; tagging at the final fixed state is more accurate
- **Public wrapper audit checklist** — when adding a public wrapper to satisfy the dispatcher: (1) add --help handler, (2) add BATS test, (3) audit all existing call sites and update to the public name

## Theme

v1.0.5 was a two-PR milestone. The code landed cleanly in PR #62 but the release process broke — Copilot's async review was missed and the branch was merged too early. The fix-up PR (#63) addressed 13 Copilot findings across two rounds and tightened the public/private contract for `acg_extend_playwright`. The milestone's real output was not the antigravity decoupling (that was straightforward) but the process improvement: a memory rule that prevents the premature merge pattern from recurring.
