# Retrospective — v1.4.7

**Date:** 2026-05-18
**Milestone:** Keycloak cross-cluster reachability + make up reliability
**PR:** #76 — merged to main (`0278e8d7076b46b79a595f8530d58d7b65208084`)
**Participants:** Claude, Codex, Haiku

## What Went Well
- Identified root cause of Keycloak unreachability quickly (wrong node IP from `_node_ip` selecting agent instead of control-plane)
- CoreDNS Python merge approach cleanly handles the one-hosts-block-per-server-block constraint
- `timeout 60` wrapper on `acg_extend_playwright` prevents indefinite hang during `make up`

## What Went Wrong
- `_node_ip` had no control-plane label selector — picked agent node (10.0.1.14) instead of control-plane (10.0.1.144); DNAT had 0 packet matches
- CoreDNS CrashLoopBackOff on each cluster rebuild due to duplicate `hosts {}` block insertion
- Realm JSON path in `bin/acg-up` pointed to `identity/config/` instead of `identity/keycloak/`
- `acg_extend_playwright` could hang indefinitely (no timeout) — blocked `make up` for >2 minutes

## Process Rules Added
- `bin/acg-up` node IP selection must always use `-l 'node-role.kubernetes.io/control-plane'` label selector
- CoreDNS custom entries must merge into existing NodeHosts block, never add a second `hosts {}` block
- Wrap Playwright calls in `timeout <N>` to prevent indefinite hangs in automated scripts

## Decisions Made
- Products table schema owned exclusively by SQLAlchemy (`create_all()`); init ConfigMap must never CREATE TABLE products
- Keycloak cross-cluster path: Pod → CoreDNS → iptables DNAT (control-plane) → SSH reverse tunnel → Mac:80 → Istio → Keycloak

## Theme
v1.4.7 fixed a cascade of make-up failures encountered when spinning up a new ACG AWS sandbox after the old one expired. The core issue was the Keycloak cross-cluster reachability chain breaking silently: a wrong node IP caused DNAT to point at the wrong host, CoreDNS entries caused CrashLoopBackOff, and the products page returned 500s due to a schema owned by the wrong layer. Each fix was surgical — timeout wrapper, control-plane label selector, Python-based CoreDNS merge, and SQLAlchemy schema ownership enforced via Codex spec.
