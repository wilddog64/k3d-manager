## Finding 1 — `scripts/plugins/acg.sh:426` — Public wrapper missing `--help` (BUG)

**What Copilot flagged:** `acg_extend_playwright` had no `--help` handler — calling it with `--help` would treat the flag as a sandbox URL and attempt to launch Playwright.

**Fix applied:**
```bash
# Before:
function acg_extend_playwright() {
  _acg_extend_playwright "${@}"
}

# After:
function acg_extend_playwright() {
  if [[ "${1:-}" == "-h" || "${1:-}" == "--help" ]]; then
    cat <<'HELP'
Usage: acg_extend_playwright <sandbox_url>
...
HELP
    return 0
  fi
  _acg_extend_playwright "${@}"
}
```

**Root cause:** The public wrapper was added to satisfy the dispatcher contract (PR #62 fix) but was not given the standard `--help` pattern that all other `acg_*` public functions implement.

**Process note:** Every new public function exposed via the dispatcher must have a `--help` handler. Check sibling functions in the same plugin before declaring a wrapper complete.

## Finding 2 — `scripts/plugins/acg.sh:425` — No BATS coverage for `acg_extend_playwright`

**What Copilot flagged:** `acg.bats` already tests `--help` for `acg_extend`, `acg_provision`, `acg_status`, `acg_teardown` — but had no test for the newly added `acg_extend_playwright`.

**Fix applied:** Added to `scripts/tests/plugins/acg.bats`:
```bash
@test "acg_extend_playwright prints help with --help" {
  run acg_extend_playwright --help
  [ "$status" -eq 0 ]
  [[ "$output" == *"Usage: acg_extend_playwright"* ]]
}
```

## Root Cause (meta)

PR #63 was itself a fix-up for PR #62 Copilot findings. The new wrapper `acg_extend_playwright` was added as a thin public shim without applying the full standard that other `acg_*` functions follow (--help handler + BATS test). The fix also exposed that callers outside the dispatcher (k3s-aws.sh, acg_watch) were still calling the private implementation directly despite a public contract being available.

**Process note:** When adding a public wrapper to satisfy the dispatcher contract, immediately audit: (1) --help handler, (2) BATS test, (3) all call sites that can now use the public name.
