## Problem

The Slack command surface predates the v1.7.0 move to a permanent Hostinger app cluster. Two gaps:

1. **No Hostinger status from Slack.** `/acg-status` reports the ACG tunnel/AWS world; the Hostinger
   app cluster (which has its own read-only `bin/hostinger-status`) has no Slack command.
2. **`/acg-up` and `/acg-down` read as "my cluster."** They are now ACG-learning-sandbox lifecycle
   only (Hostinger is permanent and provisioned out-of-band). Their Slack copy should say so.

**Decision (user, 2026-06-14):** keep `/acg-up` + `/acg-down` but relabel them as ACG-lab-only;
add a **separate** `/hostinger-status` command (do NOT make `/acg-status` provider-aware).

**Root cause:** the Slack surface was built entirely around the ephemeral ACG sandbox model.

def _run_hostinger_status(job_id, response_url, thread_ts=None):
    """Run bin/hostinger-status (read-only); post result to response_url or thread."""
    job_dir = JOB_DIR / job_id
    if thread_ts:
        (job_dir / "thread_ts").write_text(thread_ts)
    lines = []

    def _log(msg):
        lines.append(msg + "\n")

    def _finish(status):
        (job_dir / "status").write_text(status)
        output = "".join(lines)
        (job_dir / "output").write_text(output)
        if response_url:
            _slack_post(response_url, output)
        else:
            _notify_job(job_id, output)

    try:
        (job_dir / "status").write_text("running")
        import shlex as _shlex_hs
        repo_root = Path(__file__).resolve().parent.parent
        status_bin = repo_root / "bin" / "hostinger-status"
        status_out, status_timeout = _posix_spawn_capture(
            ["/bin/bash", "-c",
             f"{_shlex_hs.quote(str(status_bin))} && echo __WEBHOOK_SUCCESS__"],
            timeout=60,
        )
        if status_timeout:
            _log("❌ *hostinger-status timed out* after 60s — run `bin/hostinger-status` manually")
            _finish("failed")
            return
        report = status_out.replace("__WEBHOOK_SUCCESS__", "").rstrip()
        _log("🖥️ *Hostinger app cluster status*")
        if report:
            _log("```\n{}\n```".format(report[-3500:]))
        if "__WEBHOOK_SUCCESS__" not in status_out:
            _log("⚠️ hostinger-status exited non-zero")
        _finish("success")
    except Exception as exc:
        _log(f"ERROR: {exc}")
        _finish("failed")


def _run_analyze(job_id, alert_payload, response_url=""):
