# Bugfix: v1.4.6 — acg_credentials does not dismiss "Session extended" modal

**Branch:** `k3d-manager-v1.4.6`
**Files:** `scripts/lib/acg/playwright/acg_credentials.js`

---

## Problem

After `acg_extend` successfully extends the sandbox, a "Session extended / Your sandbox has been
extended." modal appears in the browser. When `acg_credentials.js` runs immediately after, the
modal may still be open, obscuring the "Start Sandbox" / "Open Sandbox" buttons and credential
input fields. This causes the Start/Open flow to fail or extract from the wrong element.

**Root cause:** `acg_credentials.js` has no modal-dismissal step before the Start/Open flow.
`acg_extend.js` has this logic (lines 140–156) but `acg_credentials.js` does not.

---

## Reproduction

1. Run `make up` (ACG extend runs and extends the session — modal appears)
2. `acg_credentials.js` runs while the modal is still visible
3. Modal obscures the sandbox panel controls

---

## Fix

### Change 1 — `scripts/lib/acg/playwright/acg_credentials.js`: dismiss "Session extended" modal before Start/Open flow

Insert a modal-dismissal block immediately before the `// 3. Handle Sandbox Start/Open Flow`
comment at line 338. This is the same pattern used in `acg_extend.js` lines 140–156.

**Exact old block (lines 337–339):**

```javascript

    // 3. Handle Sandbox Start/Open Flow
    // Skip only if credentials are already populated (not just visible — inputs render empty before start)
```

**Exact new block:**

```javascript

    // Dismiss any lingering "Session extended" modal that may obscure sandbox controls
    const _sessionExtendedModal = page.locator('text="Session extended"').first();
    if (await _sessionExtendedModal.isVisible({ timeout: 3000 }).catch(() => false)) {
      console.error('INFO: Dismissing "Session extended" modal...');
      await page.keyboard.press('Escape').catch(() => {});
      await page.waitForTimeout(500);
      const _closeBtn = page.locator('[role="dialog"] button, button:has-text("×"), button[aria-label*="close" i]').first();
      if (await _sessionExtendedModal.isVisible({ timeout: 1000 }).catch(() => false) &&
          await _closeBtn.isVisible({ timeout: 1000 }).catch(() => false)) {
        await _closeBtn.click({ force: true }).catch(() => {});
        await page.waitForTimeout(500);
      }
      await _sessionExtendedModal.waitFor({ state: 'hidden', timeout: 3000 }).catch(() => {
        console.error('WARN: "Session extended" modal did not close within 3s — proceeding anyway');
      });
    }

    // 3. Handle Sandbox Start/Open Flow
    // Skip only if credentials are already populated (not just visible — inputs render empty before start)
```

Only these lines change. Do NOT modify any other lines in the file.
