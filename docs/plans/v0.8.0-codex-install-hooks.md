# v0.8.0: Install tracked pre-commit hook on all machines

## Branch

`k3d-manager-v0.8.0`

## Problem

The tracked pre-commit hook at `scripts/hooks/pre-commit` is not wired up
consistently across machines:

- **M4 Air** (main dev): `.git/hooks/pre-commit` is a stale copy missing the
  subtree guard and `_agent_lint` check — not in sync with the tracked version
- **M2 Air** (`m2jump`): no hook installed — Gemini committed without `_agent_audit` running
- **Ubuntu** (`ubuntu`): unknown — check and install if k3d-manager repo is present

The hook must be a **symlink** so it always uses the current tracked version
without manual updates.

---

## Task

### Step 1 — Create `scripts/hooks/install-hooks.sh`

```bash
#!/usr/bin/env bash
# Install tracked git hooks as symlinks into .git/hooks/
set -euo pipefail

REPO_ROOT="$(git rev-parse --show-toplevel)"
HOOKS_SRC="$REPO_ROOT/scripts/hooks"
HOOKS_DST="$REPO_ROOT/.git/hooks"

for hook in "$HOOKS_SRC"/*; do
   [[ -f "$hook" ]] || continue
   hook_name="$(basename "$hook")"
   target="$HOOKS_DST/$hook_name"
   if [[ -L "$target" ]]; then
      echo "[hooks] $hook_name already a symlink — skipping"
   else
      ln -sf "../../scripts/hooks/$hook_name" "$target"
      echo "[hooks] installed: $hook_name"
   fi
done
```

### Step 2 — Install on M4 Air (local)


Verify the symlink was created:
