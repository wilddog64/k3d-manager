## Problem

`acg_get_credentials` calls `_ensure_antigravity`, `_antigravity_launch`, and
`_antigravity_ensure_acg_session` — all defined in `scripts/plugins/antigravity.sh`.

`acg.sh` does not source `antigravity.sh`. When `acg.sh` is loaded without `antigravity.sh`
already loaded (e.g., via a direct dispatcher call), these functions are undefined and fail
with "command not found".

Discovered during Gemini e2e smoke test (2026-03-29).

---

## Fix

Add `source "${SCRIPT_DIR}/plugins/antigravity.sh"` to the source block at the top of `acg.sh`,
after the existing `aws.sh` source.
