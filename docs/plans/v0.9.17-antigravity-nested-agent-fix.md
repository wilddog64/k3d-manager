# Spec — v0.9.17: Fix Nested Gemini Agent — Plan Mode + Path Restrictions

**Branch:** `k3d-manager-v0.9.17`
**Implementation:** Gemini
**Tests:** Codex (BATS)
**Motivation:** E2E test failed because the inner `gemini` subprocess runs in Plan Mode by

`gemini` invocation, and change all `/tmp/ag_*.js` paths to the allowed workspace temp dir.

## Implementation (Gemini)

### Target file

`scripts/plugins/antigravity.sh`

### Change 1 — Add `--approval-mode yolo` to gemini invocation

In `_antigravity_gemini_prompt` at line 29, change:

```bash
    output=$(gemini --model "$model" --prompt "$prompt" 2>&1)
```

To:

```bash
    output=$(gemini --model "$model" --approval-mode yolo --prompt "$prompt" 2>&1)
```

### Change 2 — Fix temp file paths in all 4 prompt strings

The inner agent cannot write to `/tmp/`. Use the allowed workspace temp dir instead.
Change every `/tmp/ag_*.js` path in the prompt strings as follows:

| Old path | New path |
|---|---|
| `/tmp/ag_github_session.js` (line 92) | `${HOME}/.gemini/tmp/k3d-manager/ag_github_session.js` |
| `/tmp/ag_acg_session.js` (line 116) | `${HOME}/.gemini/tmp/k3d-manager/ag_acg_session.js` |
| `/tmp/ag_trigger.js` (line 161) | `${HOME}/.gemini/tmp/k3d-manager/ag_trigger.js` |
| `/tmp/ag_acg_extend.js` (line 209) | `${HOME}/.gemini/tmp/k3d-manager/ag_acg_extend.js` |

Note: these paths are inside double-quoted strings — `${HOME}` will expand correctly at runtime.

## Tests (Codex — after Gemini pushes)

### Target file

`scripts/tests/lib/antigravity.bats`

### Add these two tests

```bash
@test "_antigravity_gemini_prompt: passes --approval-mode yolo to gemini" {
  _info() { :; }
  gemini() {
    local found_yolo=0
    local i
    for i in "$@"; do
      [[ "$i" == "yolo" ]] && found_yolo=1
    done
    [[ $found_yolo -eq 1 ]] || { echo "missing --approval-mode yolo: $*"; return 1; }
    echo "ok"
    return 0
  }
  export -f _info gemini
  source scripts/plugins/antigravity.sh

  run _antigravity_gemini_prompt "test prompt"
  [ "$status" -eq 0 ]
  [[ "$output" == *"ok"* ]]
  unset -f _info gemini
}

@test "_antigravity_gemini_prompt: creates workspace temp dir" {
  _info() { :; }
  gemini() { echo "ok"; return 0; }
  export -f _info gemini
  source scripts/plugins/antigravity.sh

  local tmpdir="${HOME}/.gemini/tmp/k3d-manager"
  rm -rf "$tmpdir"
  run _antigravity_gemini_prompt "test prompt"
  [ "$status" -eq 0 ]
  [ -d "$tmpdir" ]
  unset -f _info gemini
}
```
