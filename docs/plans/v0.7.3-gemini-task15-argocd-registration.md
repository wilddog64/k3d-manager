## Task 1 — Ubuntu Kubeconfig Prerequisite (run once on Ubuntu)

This copies the k3s kubeconfig to a user-readable location so SSH export works.

```bash
ssh ubuntu "mkdir -p /home/parallels/.kube && \
  sudo cp /etc/rancher/k3s/k3s.yaml /home/parallels/.kube/k3s.yaml && \
  sudo chown parallels:parallels /home/parallels/.kube/k3s.yaml && \
  chmod 600 /home/parallels/.kube/k3s.yaml"

# Verify
ssh ubuntu "ls -la /home/parallels/.kube/k3s.yaml"
# Expected: -rw------- 1 parallels parallels ...
```

**STOP — confirm file exists before continuing.**

## Task 3 — Add GitHub SSH Key to ArgoCD

Port-forward ArgoCD and log in:
```bash
nohup kubectl port-forward svc/argocd-server -n cicd 8080:443 \
  > /tmp/argocd-pf.log 2>&1 &
sleep 5
argocd login localhost:8080 --insecure
```

Add the GitHub SSH credential:
```bash
argocd repo add git@github.com:wilddog64/shopping-cart-infra \
  --ssh-private-key-path ~/.ssh/id_ed25519

argocd repo list
# Expected: git@github.com:wilddog64/shopping-cart-infra  Successful
```

**STOP — report `argocd repo list` output.**
