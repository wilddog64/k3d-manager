## Overview

Pull the `wilddog64/lib-acg` repo as a git subtree at `scripts/lib/acg/`, then:

- Replace `scripts/plugins/acg.sh` and `scripts/plugins/gcp.sh` with one-line stubs.
- Wire `scripts/plugins/gemini.sh` to source `_browser_launch` and `_cdp_ensure_acg_session`
  from the subtree instead of defining them locally.
- Update `scripts/tests/plugins/aws.bats` stubs for the renamed function.

## Step 2 — Replace `scripts/plugins/acg.sh` with stub

Replace the entire file content with:

```bash
#!/usr/bin/env bash
# scripts/plugins/acg.sh — stub: delegates to lib-acg subtree
# shellcheck source=/dev/null
source "${SCRIPT_DIR}/plugins/aws.sh"
# shellcheck source=/dev/null
source "${SCRIPT_DIR}/lib/acg/scripts/plugins/acg.sh"
```

`aws.sh` must be sourced first because lib-acg's `acg.sh` calls `aws_import_credentials`
but does not source `aws.sh` itself (it relies on the k3d-manager caller to provide it).
