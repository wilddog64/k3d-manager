# Spec — v0.9.17: Live E2E Test — `_antigravity_ensure_acg_session`

**Branch:** `k3d-manager-v0.9.17`
**Agent:** Gemini (live browser required — M4 Air only)
**Type:** Verification only — NO code changes, NO commits
**Scope:** Confirm `_antigravity_ensure_acg_session` behaves correctly in both scenarios

## Before You Start

1. `git pull origin k3d-manager-v0.9.17`
2. Confirm Antigravity browser is running with `--remote-debugging-port=9222`
3. Have an ACG sandbox URL ready — this must be provided by the user. Do NOT provision a sandbox, do NOT run `acg.sh`, do NOT use AWS CLI or AWS credentials. If no sandbox URL has been provided, stop and ask.

---

## Test Scenario 1 — Session Already Active (happy path)

**Setup:** Ensure you are already logged into ACG in the Antigravity browser.

**Run:**
```bash
cd ~/src/gitrepo/personal/k3d-manager
source scripts/lib/system.sh
source scripts/plugins/antigravity.sh
antigravity_acg_extend "<your-acg-sandbox-url>"
```

**Expected:**
- `_antigravity_ensure_acg_session` fires and detects the existing session
- Output contains `ACG_SESSION_OK`
- No login prompt appears
- Execution continues to `_info "Extending ACG sandbox TTL..."` and completes normally

**Report:** Paste the full terminal output.

---

## Test Scenario 2 — Session Expired (login required)

**Setup:** Log out of ACG in the Antigravity browser (or clear ACG cookies).

**Run:**
```bash
cd ~/src/gitrepo/personal/k3d-manager
source scripts/lib/system.sh
source scripts/plugins/antigravity.sh
antigravity_acg_extend "<your-acg-sandbox-url>"
```

**Expected:**
- `_antigravity_ensure_acg_session` fires and detects no session
- Output contains: `ACTION REQUIRED: Please log into ACG in the Antigravity browser window, then press Enter to continue.`
- After logging in through the browser, execution continues automatically
- Output contains `ACG_SESSION_OK` after login
- Execution continues to `_info "Extending ACG sandbox TTL..."` and completes normally

**Report:** Paste the full terminal output. If you cannot log out of ACG easily, skip Scenario 2 and note it.
