# v1.0.3 — Remove CDP probe from `acg_credentials.js`

**Date**: 2026-04-03
**Branch**: k3d-manager-v1.0.3
**Status**: ASSIGNED to Codex

## Problem

`acg_credentials.js` has a CDP probe that tries to reuse an existing Chrome window via
`chromium.connectOverCDP('http://localhost:9222')`. This adds complexity and has caused two
`disconnect()`/`close()` bugs already. It is unnecessary because `launchPersistentContext`
with a persisted auth dir (`~/.local/share/k3d-manager/playwright-auth`) already handles
session persistence — Playwright remembers the Pluralsight login across runs without
needing to attach to a running Chrome window.

`acg.sh`'s `acg_get_credentials` also has a pre-check that launches Chrome via `_browser_launch`
if CDP port 9222 is not available. With `launchPersistentContext`, Chrome is launched by the
Playwright script itself — this pre-check is no longer needed.

## Files Changed

1. `scripts/playwright/acg_credentials.js` — remove CDP probe block + simplify `finally`
2. `scripts/plugins/acg.sh` — remove CDP pre-check block in `acg_get_credentials`

## Change 1 — `scripts/playwright/acg_credentials.js`

### 1a. Replace CDP probe + conditional launch with direct `launchPersistentContext`

**Old** (lines 39–69):
```javascript
  let browserContext;
  let _cdpBrowser = null;
  try {
    try {
      _cdpBrowser = await chromium.connectOverCDP('http://localhost:9222');
      const _cdpContexts = _cdpBrowser.contexts();
      if (_cdpContexts.length > 0) {
        const _cdpContext = _cdpContexts[0];
        const _cdpPages = _cdpContext.pages();
        const _cdpPsPage = _cdpPages.find(p => {
          try { return new URL(p.url()).hostname.endsWith('.pluralsight.com'); } catch { return false; }
        });
        if (_cdpPsPage) {
          console.error('INFO: Found existing Pluralsight session via CDP — reusing existing Chrome instance.');
          browserContext = _cdpContext;
        }
      }
      if (!browserContext) {
        try { await _cdpBrowser.close(); } catch {}
        _cdpBrowser = null;
      }
    } catch {
      _cdpBrowser = null;
    }
    if (!browserContext) {
      browserContext = await chromium.launchPersistentContext(AUTH_DIR, {
        headless: false,
        channel: 'chrome',
        args: ['--password-store=basic'],
      });
    }
```

**New**:
```javascript
  let browserContext;
  try {
    browserContext = await chromium.launchPersistentContext(AUTH_DIR, {
      headless: false,
      channel: 'chrome',
      args: ['--password-store=basic'],
    });
```

### 1b. Simplify `finally` block

**Old** (lines 296–302):
```javascript
  } finally {
    if (_cdpBrowser) {
      try { await _cdpBrowser.close(); } catch {}
    } else if (browserContext) {
      await browserContext.close();
    }
  }
```

**New**:
```javascript
  } finally {
    if (browserContext) {
      await browserContext.close().catch(() => {});
    }
  }
```
