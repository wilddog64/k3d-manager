# Spec: Fix agent_rigor.sh IP Allowlist — Upstream First

## Problem

Codex's `abe149f` introduced an IP literal allowlist in `scripts/lib/agent_rigor.sh`
directly (k3d-manager working copy). Two violations:

1. **Hardcoded path**: allowlist path is `${SCRIPT_DIR}/etc/agent/hardcoded-ip-allowlist` —
   specific to k3d-manager's layout; cannot be adopted by other consumers of lib-foundation.
2. **Subtree discipline**: `scripts/lib/agent_rigor.sh` is managed from lib-foundation.
   Edits must go upstream first; k3d-manager's copy is synced from
   `scripts/lib/foundation/scripts/lib/agent_rigor.sh` (the subtree snapshot).

The CF template (`scripts/etc/acg-cluster.yaml`) contains CIDR literals that legitimately
trip the IP audit. The allowlist feature is correct — but the implementation must be
upstreamed so all consumers can use it.

## Fix

Two repos. Codex does both in one commit per repo.

---

## Repo 1 — lib-foundation

**Branch:** `feat/v0.3.15`
**Path:** `~/src/gitrepo/personal/lib-foundation/`
**File:** `scripts/lib/agent_rigor.sh`

Replace the IP audit loop to support the `AGENT_IP_ALLOWLIST` env var.
Consumers set this env var before sourcing `agent_rigor.sh`; lib-foundation
reads it — no hardcoded paths.

### Old code (lines 175–185 in lib-foundation)

```bash
   local file
   while IFS= read -r -d '' file; do
      local ip_lines
      ip_lines=$(git show :"$file" 2>/dev/null \
         | grep -En '[0-9]{1,3}\.[0-9]{1,3}\.[0-9]{1,3}\.[0-9]{1,3}' || true)
      if [[ -n "$ip_lines" ]]; then
         _warn "Agent audit: hardcoded IP address in $file — use a CoreDNS hostname instead:"
         _warn "$ip_lines"
         status=1
      fi
   done < <(git diff --cached --name-only --diff-filter=ACM -z -- '*.yaml' '*.yml' 2>/dev/null || true)
```

### New code

```bash
   local file
   while IFS= read -r -d '' file; do
      if [[ -n "${AGENT_IP_ALLOWLIST:-}" && -r "${AGENT_IP_ALLOWLIST}" ]]; then
         local skip_file=0
         while IFS= read -r allow || [[ -n "$allow" ]]; do
            [[ -z "$allow" || "$allow" =~ ^# ]] && continue
            if [[ "$file" == "$allow" ]]; then
               skip_file=1
               break
            fi
         done < "${AGENT_IP_ALLOWLIST}"
         [[ "$skip_file" -eq 1 ]] && continue
      fi
      local ip_lines
      ip_lines=$(git show :"$file" 2>/dev/null \
         | grep -En '[0-9]{1,3}\.[0-9]{1,3}\.[0-9]{1,3}\.[0-9]{1,3}' || true)
      if [[ -n "$ip_lines" ]]; then
         _warn "Agent audit: hardcoded IP address in $file — use a CoreDNS hostname instead:"
         _warn "$ip_lines"
         status=1
      fi
   done < <(git diff --cached --name-only --diff-filter=ACM -z -- '*.yaml' '*.yml' 2>/dev/null || true)
```
