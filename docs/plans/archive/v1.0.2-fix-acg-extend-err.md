# v1.0.2 — Fix `antigravity_acg_extend` Fatal Exit on Extend Failure

**Date**: 2026-04-03
**Branch**: k3d-manager-v1.0.2
**Status**: COMPLETE (applied directly by Claude)
**Issue**: `docs/issues/2026-04-03-antigravity-acg-extend-err-exits-process.md`

## Problem

`antigravity_acg_extend` calls `_err` when the Playwright extend script fails. `_err` calls
`exit 1`, killing the entire shell process. The caller in `k3s-aws.sh` handles failure with
`|| _info "...proceeding"`, but that handler never runs because the process is dead.

## Change

**File**: `scripts/plugins/antigravity.sh` — `antigravity_acg_extend` (lines 224–226)

Old:
```bash
if [[ $exit_code -ne 0 ]]; then
  _err "[antigravity] acg_extend failed: ${output}"
fi
```

New:
```bash
if [[ $exit_code -ne 0 ]]; then
  _info "[antigravity] acg_extend failed: ${output}"
  return 1
fi
```
