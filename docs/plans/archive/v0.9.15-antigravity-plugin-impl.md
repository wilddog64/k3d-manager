# Spec: Antigravity Plugin Implementation (v0.9.15)

## Goal

Rewrite `scripts/plugins/antigravity.sh` to use `gemini` CLI (`@google/gemini-cli`) as the
automation engine. Gemini drives Playwright for browser interaction (GitHub Copilot agent
trigger, ACG TTL extend) and uses its built-in `web_fetch` for reading task output.

The existing skeleton has wrong assumptions (`agy` as automation CLI). Replace it entirely.

## Implementation

### Private helpers

#### `_ensure_antigravity`

Check for `gemini` binary. Install via `npm install -g @google/gemini-cli` if missing.
Requires Node.js — call `_ensure_node` first.

```bash
function _ensure_antigravity() {
  if _command_exist gemini; then
    return 0
  fi
  _log "gemini CLI not found — installing @google/gemini-cli..."
  _ensure_node
  _run_command -- npm install -g @google/gemini-cli
  if ! _command_exist gemini; then
    _err "gemini CLI install succeeded but binary not found in PATH"
  fi
}
```

#### `_ensure_playwright`

Check for `playwright` binary. Install via npm if missing, then install Chromium browser.

```bash
function _ensure_playwright() {
  if _command_exist playwright; then
    return 0
  fi
  _log "Playwright not found — installing..."
  _ensure_node
  _run_command -- npm install -g playwright
  _run_command -- playwright install chromium
}
```

#### `antigravity_trigger_copilot_review`

**Signature:** `antigravity_trigger_copilot_review <owner> <repo> [review_prompt]`

**What it does:** Passes a structured prompt to `gemini --prompt` instructing it to write
and run a Playwright script that navigates to `github.com/<owner>/<repo>/agents`, triggers
a new Copilot coding agent task, waits for the task URL to appear, and prints the task UUID.

**Prompt template (pass verbatim to `gemini --prompt`):**

```
You are a browser automation agent. Use Playwright (Node.js) to do the following:

1. Launch a Chromium browser (headless: false for first run, headless: true otherwise).
2. Navigate to https://github.com/<owner>/<repo>/agents
3. Authenticate if prompted (use stored GitHub session cookies if available).
4. Find and click the button to create a new Copilot coding agent task.
5. Enter this review prompt in the task input field:
   "<review_prompt>"
6. Submit the task.
7. Wait until the page URL changes to https://github.com/<owner>/<repo>/tasks/<uuid>
8. Extract and print ONLY the task UUID (the last path segment of the URL).

Write the Playwright script to a temp file, execute it with node, and print the UUID.
If the task UUID is not found within 60 seconds, exit with code 1.
```

**Implementation:**

```bash
function antigravity_trigger_copilot_review() {
  local owner="${1:?usage: antigravity_trigger_copilot_review <owner> <repo> [prompt]}"
  local repo="${2:?usage: antigravity_trigger_copilot_review <owner> <repo> [prompt]}"
  local review_prompt="${3:-Review this codebase for code quality, security, and architecture.}"

  _ensure_antigravity
  _ensure_playwright

  _log "Triggering Copilot coding agent on ${owner}/${repo}..."

  local gemini_prompt
  gemini_prompt="You are a browser automation agent. Use Playwright (Node.js) to do the following:

1. Launch a Chromium browser (headless: true).
2. Navigate to https://github.com/${owner}/${repo}/agents
3. Find and click the button to create a new Copilot coding agent task.
4. Enter this review prompt in the task input: ${review_prompt}
5. Submit the task.
6. Wait until the page URL changes to https://github.com/${owner}/${repo}/tasks/<uuid>.
7. Extract and print ONLY the task UUID (last path segment of the URL).

Write the Playwright script to /tmp/ag_trigger.js, execute with node, print the UUID.
Exit code 1 if UUID not found within 60 seconds."

  gemini --prompt "$gemini_prompt"
}
```

#### `antigravity_poll_task`

**Signature:** `antigravity_poll_task <owner> <repo> <task_uuid> [timeout_seconds]`

**What it does:** Passes a prompt to `gemini --prompt` instructing it to use `web_fetch`
to poll the task page until status is complete, then extract and print the full review output.

**Prompt template:**

```
Use your web_fetch tool to fetch https://github.com/<owner>/<repo>/tasks/<task_uuid>
Repeat every 30 seconds until the task status shows "complete" or "done".
Timeout after <timeout> seconds — exit with error if not complete by then.
Once complete, extract and print the full review text from the page.
Do not summarize — print the complete output verbatim.
```

**Implementation:**

```bash
function antigravity_poll_task() {
  local owner="${1:?usage: antigravity_poll_task <owner> <repo> <task_uuid> [timeout]}"
  local repo="${2:?usage: antigravity_poll_task <owner> <repo> <task_uuid> [timeout]}"
  local task_uuid="${3:?usage: antigravity_poll_task <owner> <repo> <task_uuid> [timeout]}"
  local timeout="${4:-300}"

  _ensure_antigravity

  _log "Polling task ${task_uuid} on ${owner}/${repo} (timeout: ${timeout}s)..."

  local gemini_prompt
  gemini_prompt="Use your web_fetch tool to fetch https://github.com/${owner}/${repo}/tasks/${task_uuid}
Poll every 30 seconds until the task status shows complete or done.
Timeout after ${timeout} seconds — if not complete by then, print ERROR: timeout and exit.
Once complete, extract and print the full review output verbatim. Do not summarize."

  gemini --prompt "$gemini_prompt"
}
```

#### `antigravity_acg_extend`

**Signature:** `antigravity_acg_extend <sandbox_url> [hours]`

**What it does:** Passes a prompt to `gemini --prompt` instructing it to write and run a
Playwright script that navigates to the ACG sandbox URL, logs in if needed, and extends
the sandbox TTL by the specified number of hours.

**Implementation:**

```bash
function antigravity_acg_extend() {
  local sandbox_url="${1:?usage: antigravity_acg_extend <sandbox_url> [hours]}"
  local hours="${2:-4}"

  _ensure_antigravity
  _ensure_playwright

  _log "Extending ACG sandbox TTL by ${hours}h at ${sandbox_url}..."

  local gemini_prompt
  gemini_prompt="You are a browser automation agent. Use Playwright (Node.js) to do the following:

1. Launch a Chromium browser (headless: true).
2. Navigate to ${sandbox_url}
3. If a Google login page appears, use stored session cookies or prompt the user to authenticate.
4. Find the sandbox TTL extend button (look for 'Extend', '+4 hours', or similar).
5. Click to extend by ${hours} hours.
6. Confirm the new TTL is shown on the page.
7. Print the new sandbox expiry time.

Write the Playwright script to /tmp/ag_acg_extend.js, execute with node, print the result.
Exit code 1 if the extend button is not found or the action fails."

  gemini --prompt "$gemini_prompt"
}
```
