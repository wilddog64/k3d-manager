## Background

`antigravity_acg_extend` currently navigates to the ACG sandbox page without verifying the
user is logged in. If the session has expired the automation silently fails.

The fix is the same pattern already proven for GitHub: `_antigravity_ensure_github_session`
connects to the running Antigravity browser via CDP, checks login state, and waits for the
user to authenticate if needed. No file I/O, no JSON state, no new browser instances.
Antigravity is a persistent headed browser — the ACG session cookies are already in it.

---

## Change

### 1. Add `_antigravity_ensure_acg_session` (private helper)

Mirror `_antigravity_ensure_github_session` exactly, targeting `learn.acloud.guru` instead
of `github.com`.

The Gemini prompt must instruct the Playwright script (`/tmp/ag_acg_session.js`) to:
1. Connect via CDP: `chromium.connectOverCDP('http://localhost:9222')`
2. Use the first browser context and page — do NOT launch a new browser
3. Navigate to `https://learn.acloud.guru` and wait for load
4. Check if logged in — look for user avatar, account menu, or dashboard elements
5. If logged in: print `ACG_SESSION_OK` and exit 0
6. If NOT logged in:
   a. Navigate to `https://learn.acloud.guru/sign-in`
   b. Print: `ACTION REQUIRED: Please log into ACG in the Antigravity browser, then press Enter`
   c. Poll every 5s until URL no longer contains `/sign-in` — timeout 300s
   d. On success: print `ACG_SESSION_OK` and exit 0
   e. On timeout: print `ERROR: ACG login timeout` and exit 1

### 2. Update `antigravity_acg_extend`

Add one line before the `_info "Extending ACG sandbox..."` line:

```bash
_antigravity_ensure_acg_session
```

That is the only change to `antigravity_acg_extend`.

## BATS Tests (Codex — after Gemini's SHA is verified)

Add to `scripts/tests/lib/antigravity.bats` (create file if it doesn't exist):

```bash
@test "_antigravity_ensure_acg_session: returns 0 when gemini succeeds" {
  _ensure_antigravity() { return 0; }
  gemini() { return 0; }
  export -f _ensure_antigravity gemini

  run _antigravity_ensure_acg_session
  [ "$status" -eq 0 ]
  unset -f _ensure_antigravity gemini
}

@test "_antigravity_ensure_acg_session: returns 1 when gemini fails" {
  _ensure_antigravity() { return 0; }
  gemini() { return 1; }
  export -f _ensure_antigravity gemini

  run _antigravity_ensure_acg_session
  [ "$status" -eq 1 ]
  unset -f _ensure_antigravity gemini
}

@test "antigravity_acg_extend: calls _antigravity_ensure_acg_session before extend" {
  session_checked=0
  _ensure_antigravity() { return 0; }
  _antigravity_launch() { return 0; }
  _antigravity_ensure_acg_session() { session_checked=1; return 0; }
  gemini() { return 0; }
  _info() { :; }
  export -f _ensure_antigravity _antigravity_launch _antigravity_ensure_acg_session gemini _info

  run antigravity_acg_extend "https://example.com/sandbox"
  [ "$session_checked" -eq 1 ]
  unset -f _ensure_antigravity _antigravity_launch _antigravity_ensure_acg_session gemini _info
}
```
