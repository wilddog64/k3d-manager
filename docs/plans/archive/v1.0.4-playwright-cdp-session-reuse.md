# v1.0.4 — Playwright CDP Session Reuse on First Run

**Date**: 2026-04-03
**Branch**: k3d-manager-v1.0.2
**Status**: Ready for Codex

## Problem

When `IS_FIRST_RUN` is true (empty `playwright-auth` dir), `acg_credentials.js` always launches
a new Chrome instance and waits for manual Pluralsight login — even when Chrome is already running
and the user is already signed in to Pluralsight. The script ignores the existing session entirely.

## Fix

On first run, probe CDP at `http://localhost:9222` before falling back to `launchPersistentContext`.
If Chrome is running and has a Pluralsight page, reuse that context directly. If not, fall through
to the existing `launchPersistentContext` flow unchanged.

Only `scripts/playwright/acg_credentials.js` changes. `acg_extend.js` is not touched.

## Changes

### `scripts/playwright/acg_credentials.js`

**Change 1 — add `_cdpBrowser` variable and CDP probe block** (replace the `let browserContext;` +
`launchPersistentContext` section):

Old:
```js
  let browserContext;
  try {
    browserContext = await chromium.launchPersistentContext(AUTH_DIR, {
      headless: false,
      channel: 'chrome',
      args: ['--password-store=basic'],
    });
```

New:
```js
  let browserContext;
  let _cdpBrowser = null;
  try {
    if (IS_FIRST_RUN) {
      try {
        _cdpBrowser = await chromium.connectOverCDP('http://localhost:9222');
        const _cdpContexts = _cdpBrowser.contexts();
        if (_cdpContexts.length > 0) {
          const _cdpContext = _cdpContexts[0];
          const _cdpPages = _cdpContext.pages();
          const _cdpPsPage = _cdpPages.find(p => {
            try { return new URL(p.url()).hostname.endsWith('.pluralsight.com'); } catch { return false; }
          });
          if (_cdpPsPage) {
            console.error('INFO: Found existing Pluralsight session via CDP — reusing existing Chrome instance.');
            browserContext = _cdpContext;
          }
        }
        if (!browserContext) {
          await _cdpBrowser.disconnect();
          _cdpBrowser = null;
        }
      } catch {
        _cdpBrowser = null;
      }
    }
    if (!browserContext) {
      browserContext = await chromium.launchPersistentContext(AUTH_DIR, {
        headless: false,
        channel: 'chrome',
        args: ['--password-store=basic'],
      });
    }
```

**Change 2 — update `finally` block to disconnect CDP instead of closing user's Chrome**:

Old:
```js
  } finally {
    if (browserContext) await browserContext.close();
  }
```

New:
```js
  } finally {
    if (_cdpBrowser) {
      await _cdpBrowser.disconnect().catch(() => {});
    } else if (browserContext) {
      await browserContext.close();
    }
  }
```
