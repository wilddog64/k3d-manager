# v1.4.6 — Auto-trust Vault CA cert in macOS System Keychain after TLS issuance

**Branch:** `k3d-manager-v1.4.6`

---

## Context

`bin/acg-up` issues a fresh Vault PKI cert on every cluster rebuild and writes the
CA cert to `${HOME}/.local/share/k3d-manager/argocd-browser-https-tls/ca.crt`.
After a rebuild the browser does not trust the new CA, so ArgoCD SSO fails with
a TLS alert until the user manually runs `sudo security add-trusted-cert`. This
should happen automatically as part of the rebuild.

---

## Goal

Immediately after `_argocd_issue_browser_tls_material` writes the CA cert, call
`security add-trusted-cert` via `_run_command --interactive-sudo --soft` so the
browser trusts the new Vault CA without any manual step.

## Changes

### `bin/acg-up`

Find the exact block (around line 280):

**Old:**
```bash
    _argocd_issue_browser_tls_material "${_argocd_browser_tls_dir}" || exit 1
    _argocd_write_browser_https_wrapper "${_argocd_browser_wrapper}" "${_argocd_browser_log}" "$(command -v socat)" "$(command -v curl)" "127.0.0.1" "${ARGOCD_BROWSER_PORT:-443}" "127.0.0.1" "8080" "${_argocd_browser_tls_cert}" "${_argocd_browser_tls_key}" "https://127.0.0.1:${ARGOCD_BROWSER_PORT:-443}/healthz"
```

**New:**
```bash
    _argocd_issue_browser_tls_material "${_argocd_browser_tls_dir}" || exit 1
    if [[ "$(uname)" == "Darwin" ]]; then
      _info "[acg-up] Trusting Vault CA cert in macOS System Keychain (may prompt for sudo password)..."
      _run_command --interactive-sudo --soft -- \
        security add-trusted-cert -d -r trustRoot \
          -k /Library/Keychains/System.keychain \
          "${_argocd_browser_tls_dir}/ca.crt"
    fi
    _argocd_write_browser_https_wrapper "${_argocd_browser_wrapper}" "${_argocd_browser_log}" "$(command -v socat)" "$(command -v curl)" "127.0.0.1" "${ARGOCD_BROWSER_PORT:-443}" "127.0.0.1" "8080" "${_argocd_browser_tls_cert}" "${_argocd_browser_tls_key}" "https://127.0.0.1:${ARGOCD_BROWSER_PORT:-443}/healthz"
```

Only these lines change. Do NOT modify any other lines in the file.
