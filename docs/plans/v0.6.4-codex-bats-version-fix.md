# v0.6.4 — Codex Task: Fix Hardcoded BATS Version

## Context

Gemini's Linux k3s validation (Phase 5) was blocked by a 404 error when
`_ensure_bats` fell through to `_install_bats_from_source`. The default version
`1.10.0` does not exist as a GitHub release tag for `bats-core/bats-core`.

Latest confirmed release: `v1.13.0`. Gemini's workaround: `BATS_PREFERRED_VERSION=1.11.0`.
