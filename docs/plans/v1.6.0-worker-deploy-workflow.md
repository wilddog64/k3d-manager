# v1.6.0 — Cloudflare Worker deploy workflow + secrets setup

## Context

`workers/slack-relay/` was added in `695a04d1` but has no automated deploy step.
Currently requires `wrangler deploy` to be run manually and secrets set by hand.
This spec adds:
1. A GitHub Actions workflow that deploys the Worker on every push to `main`
   when `workers/slack-relay/` changes.
2. `bin/k3dm-worker-setup` — checks whether the three required GitHub secrets exist
   and interactively prompts for any that are missing.

## File 1: `.github/workflows/deploy-worker.yml`

```yaml
---
name: Deploy Cloudflare Worker

"on":
  push:
    branches:
      - main
    paths:
      - 'workers/slack-relay/**'

permissions:
  contents: read

concurrency:
  group: deploy-worker-${{ github.ref }}
  cancel-in-progress: true

jobs:
  deploy:
    runs-on: ubuntu-latest
    steps:
      - name: Checkout
        uses: actions/checkout@v4

      - name: Deploy Slack relay Worker
        uses: cloudflare/wrangler-action@v3
        with:
          apiToken: ${{ secrets.CLOUDFLARE_API_TOKEN }}
          workingDirectory: workers/slack-relay
          secrets: |
            WEBHOOK_TOKEN
            SLACK_SIGNING_SECRET
        env:
          WEBHOOK_TOKEN: ${{ secrets.K3DM_WEBHOOK_TOKEN }}
          SLACK_SIGNING_SECRET: ${{ secrets.SLACK_SIGNING_SECRET }}
```

## File 2: `bin/k3dm-worker-setup`

Create with mode `0755`. Standalone bash — does not source lib (runs before full stack is up).

```bash
#!/usr/bin/env bash
# bin/k3dm-worker-setup — ensure GitHub secrets required for Worker deploy exist.
# Checks each secret; prompts interactively for any that are missing.

set -euo pipefail

_REPO="$(gh repo view --json nameWithOwner -q '.nameWithOwner' 2>/dev/null)" || {
  echo "[k3dm-worker-setup] ERROR: gh not authenticated — run: gh auth login"
  exit 1
}

_info()  { echo "[k3dm-worker-setup] $*"; }
_error() { echo "[k3dm-worker-setup] ERROR: $*" >&2; exit 1; }

_secret_exists() {
  gh secret list --repo "${_REPO}" --json name -q '.[].name' \
    | grep -qx "${1}" 2>/dev/null
}

_prompt_and_set() {
  local name="$1"
  local hint="$2"
  echo ""
  _info "Secret missing: ${name}"
  _info "${hint}"
  printf "[k3dm-worker-setup] Enter value (input hidden): "
  local _val
  read -rs _val
  echo ""
  [ -z "${_val}" ] && _error "Empty value — ${name} not set."
  printf '%s' "${_val}" | gh secret set "${name}" --repo "${_REPO}"
  _info "${name} set."
}

_info "Checking GitHub secrets for repo: ${_REPO}"
echo ""

_required=(
  "CLOUDFLARE_API_TOKEN:Get from: dash.cloudflare.com → My Profile → API Tokens → 'Edit Cloudflare Workers' template"
  "SLACK_SIGNING_SECRET:Get from: api.slack.com/apps → your app → Basic Information → Signing Secret"
  "K3DM_WEBHOOK_TOKEN:Generate with: python3 -c 'import secrets; print(secrets.token_hex(32))'"
)

_all_present=true
for _entry in "${_required[@]}"; do
  _name="${_entry%%:*}"
  _hint="${_entry#*:}"
  if _secret_exists "${_name}"; then
    _info "${_name}: already set ✓"
  else
    _all_present=false
    _prompt_and_set "${_name}" "${_hint}"
  fi
done

echo ""
if [ "${_all_present}" = "true" ]; then
  _info "All secrets already present — Worker deploy workflow is ready."
else
  _info "All secrets now set — Worker deploy workflow is ready."
fi
```
