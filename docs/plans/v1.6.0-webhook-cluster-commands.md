# v1.6.0 — k3dm-webhook: cluster up/down commands

## Context

Add `POST /api/v1/cluster` to `bin/k3dm-webhook` so the laptop cluster can be brought up
or torn down remotely (Slack, phone, another machine via the Cloudflare tunnel).

`KEEP_LOCAL=1` is hardcoded in the server for `action=down` — it is never accepted as a
request parameter. This guarantees the local Hub cluster cannot be destroyed via the
webhook regardless of what the caller sends.

## New Endpoint

```
POST /api/v1/cluster
Authorization: Bearer <token>
Content-Type: application/json

{"action": "up" | "down"}
```

- `action=up`   → runs `make up` in `REPO_ROOT`
- `action=down` → runs `make down KEEP_LOCAL=1` in `REPO_ROOT` (hardcoded, non-overridable)
- Any other `action` value → 400
- Missing `action` → 400
- Returns `{"job_id": "<8-hex>", "status": "queued"}` (202)
- Job output available via existing `GET /api/v1/status/<job_id>`

## Change 1: `bin/k3dm-webhook` — add `_run_cluster`

Insert the following function **after** the closing `except` block of `_run_upgrade`
(after line `_finish("failed")` in `_run_upgrade`) and **before** `class _Handler`:

```python
def _run_cluster(job_id, action):
    job_dir = JOB_DIR / job_id
    lines = []

    def _log(msg):
        lines.append(msg + "\n")

    def _finish(status):
        (job_dir / "status").write_text(status)
        (job_dir / "output").write_text("".join(lines))

    try:
        (job_dir / "status").write_text("running")
        if action == "up":
            cmd = ["make", "up"]
        else:
            cmd = ["make", "down", "KEEP_LOCAL=1"]
        _log(f"Running {' '.join(cmd)}...")
        proc = subprocess.Popen(
            cmd,
            cwd=REPO_ROOT,
            stdout=subprocess.PIPE,
            stderr=subprocess.STDOUT,
            text=True
        )
        for line in proc.stdout:
            lines.append(line)
        proc.wait()
        if proc.returncode != 0:
            raise RuntimeError(f"{' '.join(cmd)} exited {proc.returncode}")
        _log(f"{action} complete")
        _finish("success")
    except Exception as exc:
        _log(f"ERROR: {exc}")
        _finish("failed")
```

```python
    def do_POST(self):
        if not self._auth():
            self._json(401, {"error": "unauthorized"})
            return
        if self.path.startswith("/api/") and not self.path.startswith("/api/v1/"):
            self._json(400, {"error": "invalid api path"})
            return

        length = int(self.headers.get("Content-Length", 0))
        if length > MAX_BODY:
            self._json(413, {"error": "request too large"})
            return
        body = json.loads(self.rfile.read(length) or b"{}")

        if self.path == "/api/v1/argocd-upgrade":
            chart_version = body.get("chart_version", "")
            stage = body.get("stage", "")
            if not chart_version or stage not in ("acg", "infra"):
                self._json(400, {"error": "chart_version and stage (acg|infra) required"})
                return
            job_id = str(uuid.uuid4())[:8]
            (JOB_DIR / job_id).mkdir(parents=True, exist_ok=True)
            (JOB_DIR / job_id / "status").write_text("queued")
            threading.Thread(
                target=_run_upgrade,
                args=(job_id, chart_version, stage),
                daemon=True
            ).start()
            self._json(202, {"job_id": job_id, "status": "queued"})
            return

        if self.path == "/api/v1/cluster":
            action = body.get("action", "")
            if action not in ("up", "down"):
                self._json(400, {"error": "action (up|down) required"})
                return
            job_id = str(uuid.uuid4())[:8]
            (JOB_DIR / job_id).mkdir(parents=True, exist_ok=True)
            (JOB_DIR / job_id / "status").write_text("queued")
            threading.Thread(
                target=_run_cluster,
                args=(job_id, action),
                daemon=True
            ).start()
            self._json(202, {"job_id": job_id, "status": "queued"})
            return

        self._json(404, {"error": "not found"})
```

## Change 3: `scripts/tests/lib/webhook.bats` — add 5 unit tests

Append the following 5 tests **before** the `# ── Level 1` comment block (after the last

```bash
@test "POST /cluster with wrong token returns 401" {
    run curl -s -o /dev/null -w "%{http_code}" -X POST \
        -H "Authorization: Bearer wrongtoken" \
        -H "Content-Type: application/json" \
        -d '{"action":"up"}' \
        "${_WEBHOOK_URL}/api/v1/cluster"
    [ "$status" -eq 0 ]
    [ "$output" = "401" ]
}

@test "POST /cluster with action=up returns 202 and job_id" {
    run curl -s -X POST \
        -H "Authorization: Bearer ${K3DM_WEBHOOK_TOKEN}" \
        -H "Content-Type: application/json" \
        -d '{"action":"up"}' \
        "${_WEBHOOK_URL}/api/v1/cluster"
    [ "$status" -eq 0 ]
    [[ "$output" == *'"status":"queued"'* ]]
    [[ "$output" == *'"job_id"'* ]]
}

@test "POST /cluster with action=down returns 202 and job_id" {
    run curl -s -X POST \
        -H "Authorization: Bearer ${K3DM_WEBHOOK_TOKEN}" \
        -H "Content-Type: application/json" \
        -d '{"action":"down"}' \
        "${_WEBHOOK_URL}/api/v1/cluster"
    [ "$status" -eq 0 ]
    [[ "$output" == *'"status":"queued"'* ]]
    [[ "$output" == *'"job_id"'* ]]
}

@test "POST /cluster with invalid action returns 400" {
    run curl -s -o /dev/null -w "%{http_code}" -X POST \
```
