# Bugfix: v1.6.3 — auto-install acg npm deps

**Branch:** `k3d-manager-v1.6.3`
**Files:** `bin/acg-refresh`, `bin/acg-up`

---

## Problem

`bin/acg-refresh` (and `bin/acg-up`) fail with:

```
ERROR: [acg] playwright npm module not found — run: cd .../scripts/lib/acg && npm install
```

when `scripts/lib/acg/node_modules` is absent. This happens after a fresh checkout, `git
clean`, or any time the gitignored `node_modules` directory is wiped.

**Root cause:** neither `bin/acg-refresh` nor `bin/acg-up` check for the npm dependencies
before invoking playwright. The check in `scripts/lib/acg/scripts/plugins/acg.sh` (line 292)
is a hard-error that requires manual intervention. `scripts/lib/acg/` is a subtree from
lib-acg — that file cannot be changed in k3d-manager.

---

## Reproduction

```bash
rm -rf scripts/lib/acg/node_modules
bin/acg-refresh
# → ERROR: [acg] playwright npm module not found — run: cd .../scripts/lib/acg && npm install
```

---

## Fix

Add a guard in each entry-point script immediately after `REPO_ROOT` is set. The guard
runs `npm install --prefer-offline` only when `node_modules/playwright` is absent, so it is
a no-op on every normal run.

### Change 1 — `bin/acg-refresh`: add npm dep guard after REPO_ROOT

**Exact old block (lines 14–17):**

```bash
set -euo pipefail

REPO_ROOT="$(cd "$(dirname "${BASH_SOURCE[0]}")/.." && pwd)"
source "${REPO_ROOT}/scripts/lib/system.sh"
```

**Exact new block:**

```bash
set -euo pipefail

REPO_ROOT="$(cd "$(dirname "${BASH_SOURCE[0]}")/.." && pwd)"
if [[ ! -d "${REPO_ROOT}/scripts/lib/acg/node_modules/playwright" ]]; then
  npm --prefix "${REPO_ROOT}/scripts/lib/acg" install --prefer-offline 2>&1
fi
source "${REPO_ROOT}/scripts/lib/system.sh"
```

---

### Change 2 — `bin/acg-up`: add npm dep guard after REPO_ROOT

**Exact old block (lines 19–23):**

```bash

# shellcheck disable=SC2034
REPO_ROOT="$(cd "$(dirname "${BASH_SOURCE[0]}")/.." && pwd)"

SCRIPT_DIR="${REPO_ROOT}/scripts"
```

**Exact new block:**

```bash

# shellcheck disable=SC2034

if [[ ! -d "${REPO_ROOT}/scripts/lib/acg/node_modules/playwright" ]]; then
  npm --prefix "${REPO_ROOT}/scripts/lib/acg" install --prefer-offline 2>&1
fi
# shellcheck disable=SC2034
SCRIPT_DIR="${REPO_ROOT}/scripts"
```
