# Bugfix: v1.4.6 â€” acg_credentials clicks first "Open Sandbox" regardless of provider type

## Problem

When running `make up CLUSTER_PROVIDER=k3s-gcp`, the credential extractor finds the AWS sandbox
"Open Sandbox" button first (AWS cards appear before GCP cards on the listing page) and opens the
AWS sandbox panel. It then tries to parse AWS credentials as GCP service account JSON and fails:
`ERROR: Service Account Credentials is not valid JSON`.

**Root cause:** The `openButton` locator at line 411 uses `.first()` with no provider filter.
On a listing page that shows both AWS and GCP sandbox cards, the AWS card's "Open Sandbox" is
always returned first.

---

## Reproduction

1. Have an AWS sandbox already open in the browser (credentials visible)
2. Run `make up CLUSTER_PROVIDER=k3s-gcp`
3. Observed: `ERROR: Service Account Credentials is not valid JSON`
4. Expected: GCP sandbox opens, GCP credentials extracted

---

## Fix

### Change 1 â€” `scripts/lib/acg/playwright/acg_credentials.js`: select provider-specific "Open Sandbox"

In the `else if (await openButton.isVisible...)` branch, walk each button's ancestors to find
the card whose inner text matches the target provider. Fall back to `.first()` only if no
provider-specific card is found.

**Exact old block (lines 424â€“428):**


**Exact new block:**

```javascript
      } else if (await openButton.isVisible({ timeout: 5000 }).catch(() => false)) {
        let _openBtnToClick = openButton;
        if (PROVIDER !== 'aws') {
          const _providerPattern = PROVIDER === 'gcp' ? 'google\\s*cloud|gcp' : PROVIDER;
          const _allOpenBtns = page.locator('button:has-text("Open Sandbox")');
          const _btnCount = await _allOpenBtns.count();
          for (let _i = 0; _i < _btnCount; _i++) {
            const _hasProvider = await _allOpenBtns.nth(_i).evaluate((el, pattern) => {
              const re = new RegExp(pattern, 'i');
              let node = el.parentElement;
              for (let _j = 0; _j < 8; _j++) {
                if (!node) break;
                if (re.test(node.innerText || '')) return true;
                node = node.parentElement;
              }
              return false;
            }, _providerPattern).catch(() => false);
            if (_hasProvider) { _openBtnToClick = _allOpenBtns.nth(_i); break; }
          }
          if (_openBtnToClick !== openButton) {
            console.error(`INFO: Found ${PROVIDER.toUpperCase()} sandbox card â€” clicking its Open Sandbox button`);
          } else {
            console.error(`WARN: No ${PROVIDER.toUpperCase()}-specific sandbox card found â€” falling back to first Open Sandbox button`);
          }
        }
        console.error('INFO: Clicking Open Sandbox...');
        await _openBtnToClick.click();
        await page.waitForTimeout(3000);
```
