## Context

`antigravity_acg_extend()` lives in `antigravity.sh` but has no dependency on the Antigravity
IDE — it only calls `node playwright/acg_extend.js`. `acg_watch()` and `_acg_watch_write_wrapper()`
in `acg.sh` call it by name. This task moves the function into `acg.sh` as `_acg_extend_playwright`,
updates the two callers, and removes the original from `antigravity.sh`.

No behaviour changes — only relocation and rename.

## Change 1 — `scripts/plugins/acg.sh`: add `_acg_extend_playwright()`

Insert the following function **immediately before** `function acg_extend()`:

```bash
_acg_extend_playwright() {
  local sandbox_url="${1:?usage: _acg_extend_playwright <sandbox_url>}"

  local script_dir
  script_dir="$(cd "$(dirname "${BASH_SOURCE[0]}")" && pwd)"
  local playwright_script="${script_dir}/../playwright/acg_extend.js"

  if ! command -v node >/dev/null 2>&1; then
    _err "[acg] node is required — install Node.js"
  fi

  _info "[acg] Extending ACG sandbox TTL at ${sandbox_url}..."
  local output exit_code
  output=$(node "$playwright_script" "$sandbox_url" 2>&1)
  exit_code=$?

  if [[ $exit_code -ne 0 ]]; then
    _info "[acg] acg_extend failed: ${output}"
    return 1
  fi

  echo "$output"
}
```
