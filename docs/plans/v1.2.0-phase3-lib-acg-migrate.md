## Overview of Adaptations

lib-acg uses `_LIB_ACG_ROOT` (resolved from `${BASH_SOURCE[0]}`) instead of k3d-manager's
`SCRIPT_DIR`. All path references are updated accordingly.

| Old (k3d-manager) | New (lib-acg) |
|---|---|
| `source "${SCRIPT_DIR}/plugins/gemini.sh"` | `source "${_LIB_ACG_ROOT}/scripts/lib/cdp.sh"` |
| `source "${SCRIPT_DIR}/etc/playwright/vars.sh"` | `source "${_LIB_ACG_ROOT}/scripts/vars.sh"` |
| `${script_dir}/../playwright/<script>.js` | `${_LIB_ACG_ROOT}/playwright/<script>.js` |
| `"${script_dir}/../k3d-manager"` | `"${_LIB_ACG_ROOT}/../../k3d-manager"` |
| `"${SCRIPT_DIR}/etc/acg-cluster.yaml"` | `"${_LIB_ACG_ROOT}/scripts/etc/acg-cluster.yaml"` |
| `_antigravity_ensure_acg_session` | `_cdp_ensure_acg_session` |
| `source "${SCRIPT_DIR}/plugins/aws.sh"` | **remove** (provided by k3d-manager caller) |
| `SCRIPT_DIR` block (lines 7–9) in acg.sh | **remove** |
| `SCRIPT_DIR` block (lines 18–20) in gcp.sh | **remove** |
