# Spec — v0.9.16: antigravity.sh MCP refactor

**Branch:** `k3d-manager-v0.9.16`
**Target file:** `scripts/plugins/antigravity.sh`
**Commit message:** `feat(antigravity): replace headless Playwright with Antigravity IDE + MCP via CDP port 9222`

---

## Context

The previous design spawned a fresh headless Chromium via Playwright. This failed because
headless browsers have no session cookies — GitHub's `/agents` endpoint returns 404 for
unauthenticated requests.

The new design:
- Antigravity IDE (installed via `_ensure_antigravity_ide`) runs as a persistent browser
- The user logs into GitHub once inside Antigravity
- Playwright connects to that running browser via CDP on port 9222 (`connectOverCDP`)
- The Playwright MCP entry is wired into Antigravity's `mcp_config.json` via
  `_ensure_antigravity_mcp_playwright`

The three lib-foundation helpers are already available in
`scripts/lib/foundation/scripts/lib/system.sh` (pulled in v0.3.12):
- `_ensure_antigravity_ide`
- `_ensure_antigravity_mcp_playwright`
- `_antigravity_browser_ready`

## Changes

### 1. Remove `_ensure_playwright`

Delete this entire function (lines 28–36):

```bash
function _ensure_playwright() {
  if _command_exist playwright; then
    return 0
  fi
  _log "Playwright not found — installing..."
  _ensure_node
  _run_command -- npm install -g playwright
  _run_command -- playwright install chromium
}
```

Playwright is no longer installed globally — `_ensure_antigravity_mcp_playwright` configures
`npx @playwright/mcp` which runs on demand via the MCP protocol.
