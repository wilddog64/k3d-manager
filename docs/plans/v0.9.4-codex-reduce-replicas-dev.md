# Codex Task: Reduce Replicas to 1 for Dev/Test Environment (v0.9.4)

## Before You Start

1. This task spans **5 separate repos** — read all instructions before touching any file
2. Confirm you are on the correct branch in each repo (listed below)
3. All 5 changes must be committed — one commit per repo

---

## Problem

All shopping-cart services run `replicas: 2` with HPA `minReplicas: 2`. On a constrained
dev cluster (t3.medium, 4GB RAM), 10 pods simultaneously thrash the node and cause
ServiceUnavailable on the k3s API server. This is a dev/test environment — HA is not
needed. `replicas: 1` and `minReplicas: 1` reduces pod count from 10 to 5 and frees
~1.2GB of memory.
