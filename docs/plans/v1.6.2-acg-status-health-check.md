# v1.6.2 — acg-status on-demand health check + Gemini analysis

**Branch:** `k3d-manager-v1.6.2`
**Files:** `bin/k3dm-webhook`, `bin/acg-status`

**Prerequisite:** `docs/plans/v1.6.2-eso-smoke-check.md` must be committed first.
`_smoke_test_services(retries=None)` must already exist before implementing this spec.

---

## Problem

Running a smoke test + Gemini triage currently only happens 90 seconds after `make up`
(via `_run_post_provision_check`). There is no way to trigger it on demand — the user
must wait for a full `acg-up` cycle to get a fresh health report.

Two gaps:
1. The Slack `/acg-status` command shows nodes + pods + ArgoCD apps but no HTTP smoke
   results or Gemini analysis.
2. The local `bin/acg-status` bash script has no service-level health checks at all.

---

## Fix

### Change 1 — `bin/k3dm-webhook`: add smoke + Gemini triage to `_run_cluster_status()`

Insert a smoke test block after the AWS credentials section, before `_finish("success")`.
The smoke test uses `retries=1` (fast path for on-demand checks) and runs Gemini triage
for each failed service (cap at 3).

**Exact old block (end of `_run_cluster_status`, lines 1101–1115 approx):**

```python

        aws_id = subprocess.run(
            ["aws", "sts", "get-caller-identity",
             "--query", "Arn", "--output", "text"],
            capture_output=True, text=True, timeout=10

        if aws_id.returncode == 0 and aws_id.stdout.strip():

        _finish("success")
    except Exception as exc:
        _log(f"ERROR: {exc}")
        _finish("failed")
```

**Exact new block:**

```python
        # AWS credentials
        aws_id = subprocess.run(
            ["aws", "sts", "get-caller-identity",
             "--query", "Arn", "--output", "text"],
            capture_output=True, text=True, timeout=10
        )
        if aws_id.returncode == 0 and aws_id.stdout.strip():
            _log(f"\n*AWS:* ✅ `{aws_id.stdout.strip()}`")
        else:
            _log("\n*AWS:* ❌ credentials invalid or expired — run `/acg-refresh`")

        # Service smoke test (1 retry — fast path for on-demand status)
        smoke = _smoke_test_services(retries=1)
        all_smoke_ok = all(ok for _, ok, _ in smoke)
        _log(f"\n*Service smoke test:*{'  all clear ✅' if all_smoke_ok else ''}")
        for svc_name, ok, detail in smoke:
            icon = "✅" if ok else "❌"
            _log(f"  {icon} {svc_name}: {detail}")

        failed_smoke = [(name, detail) for name, ok, detail in smoke if not ok]
        for svc_name, err_detail in failed_smoke[:3]:
            diag_parts = [f"Service `{svc_name}` failed HTTP smoke test: {err_detail}"]
            ns_map = {
                "Frontend":              ("shopping-cart-apps", "app=frontend"),
                "Product images":        ("shopping-cart-apps", "app=product-catalog"),
                "Keycloak":              ("identity",            "app.kubernetes.io/name=keycloak"),
                "Prometheus":            ("monitoring",          "app.kubernetes.io/name=prometheus"),
                "Grafana":               ("monitoring",          "app.kubernetes.io/name=grafana"),
                "ArgoCD":                ("cicd",                "app.kubernetes.io/name=argocd-server"),
                "ESO ClusterSecretStore":("secrets",             "app.kubernetes.io/name=external-secrets"),
                "ESO ExternalSecrets":   ("secrets",             "app.kubernetes.io/name=external-secrets"),
            }
            ns, label_sel = ns_map.get(svc_name, ("shopping-cart-apps", ""))
            if label_sel:
                pod_state = subprocess.run(
                    ["kubectl", "get", "pods", "-n", ns, "-l", label_sel,
                     "--context", "ubuntu-k3s", "--no-headers", "--request-timeout=8s"],
                    capture_output=True, text=True, timeout=15,
                )
                if pod_state.returncode == 0 and pod_state.stdout.strip():
                    diag_parts.append(f"Pod state ({ns}):\n{pod_state.stdout.strip()[:800]}")
            prompt = (
                _ARCH_CONTEXT + "\n"
                f"Service `{svc_name}` failed an HTTP smoke test. Error: {err_detail}\n\n"
                "Diagnostics:\n" + "\n\n".join(diag_parts) + "\n\n"
                "Is this a TRANSIENT issue (pod still starting, port-forward not ready, "
                "ESO secret not yet synced — will resolve in a few minutes) or a REAL FAILURE "
                "(crash loop, missing secret, misconfiguration — needs human action)?\n"
                "Use the architecture context above to reason about upstream dependencies. "
                "Reply with TRANSIENT or REAL FAILURE, then one sentence with the most likely "
                "cause. Be brief."
            )
            analysis = _call_gemini(prompt)
            _log(f"\n🔍 *{svc_name}:* {analysis}")

        _finish("success")
    except Exception as exc:
        _log(f"ERROR: {exc}")
        _finish("failed")
```

### Change 2 — `bin/k3dm-webhook`: add `/api/v1/health` GET endpoint

The local `bin/acg-status` bash script calls this endpoint to get smoke test results
without needing Slack. It runs `_smoke_test_services(retries=1)` synchronously and returns
JSON. This endpoint is GET (not POST) and requires Bearer auth like all other `/api/v1/*`
paths.

**Exact old block (`do_GET`, lines 1348–1357 approx):**

```python
    def do_GET(self):
        if not self._auth():
            self._json(401, {"error": "unauthorized"})
            return
        if self.path.startswith("/api/") and not self.path.startswith("/api/v1/"):
            self._json(400, {"error": "invalid api path"})
            return
        if not self.path.startswith("/api/v1/status/"):
            self._json(404, {"error": "not found"})
            return
```

**Exact new block:**

```python
    def do_GET(self):
        if not self._auth():
            self._json(401, {"error": "unauthorized"})
            return
        if self.path.startswith("/api/") and not self.path.startswith("/api/v1/"):
            self._json(400, {"error": "invalid api path"})
            return
        if self.path == "/api/v1/health":
            services = _smoke_test_services(retries=1)
            result = [{"name": n, "ok": ok, "detail": d} for n, ok, d in services]
            self._json(200, {"services": result, "all_ok": all(r["ok"] for r in result)})
            return
        if not self.path.startswith("/api/v1/status/"):
            self._json(404, {"error": "not found"})
            return
```

### Change 3 — `bin/acg-status`: add Service Health section

Append a `=== Service Health ===` section at the end of `bin/acg-status`. It reads the
webhook token from the macOS Keychain (same source as `make rotate-webhook-token`) and
calls `GET /api/v1/health`. Falls back gracefully if the webhook is not running.

**Exact old block (last 2 lines of `bin/acg-status`, lines 68–70 approx):**

```bash
  if [[ "${CLUSTER_PROVIDER}" != "k3s-gcp" ]]; then
  echo ""
  echo "=== AWS Credentials ==="
  if aws sts get-caller-identity --output table 2>/dev/null; then
    : # printed inline
  else
    echo "AWS credentials invalid or expired — run bin/acg-refresh"
  fi
fi
```

**Exact new block:**

```bash
  if [[ "${CLUSTER_PROVIDER}" != "k3s-gcp" ]]; then
  echo ""
  echo "=== AWS Credentials ==="
  if aws sts get-caller-identity --output table 2>/dev/null; then
    : # printed inline
  else
    echo "AWS credentials invalid or expired — run bin/acg-refresh"
  fi
fi

echo ""
echo "=== Service Health ==="
_WEBHOOK_TOKEN="${K3DM_WEBHOOK_TOKEN:-$(security find-generic-password -s k3dm-webhook-token -w 2>/dev/null || true)}"
_WEBHOOK_PORT="${K3DM_WEBHOOK_PORT:-7443}"
if [[ -n "${_WEBHOOK_TOKEN}" ]]; then
  _health_json=$(curl -sf --max-time 35 \
    -H "Authorization: Bearer ${_WEBHOOK_TOKEN}" \
    "http://localhost:${_WEBHOOK_PORT}/api/v1/health" 2>/dev/null || true)
  if [[ -n "${_health_json}" ]]; then
    echo "${_health_json}" | python3 -c "
import sys, json
data = json.load(sys.stdin)
for s in data.get('services', []):
    icon = '✅' if s['ok'] else '❌'
    print(f'  {icon} {s[\"name\"]}: {s[\"detail\"]}')
"
  else
    echo "  (webhook unreachable at localhost:${_WEBHOOK_PORT} — run: make restart-webhook)"
  fi
else
  echo "  (webhook token not in Keychain — set K3DM_WEBHOOK_TOKEN or run: make install-webhook)"
fi
```
