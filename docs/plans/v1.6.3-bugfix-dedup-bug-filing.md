# Bugfix: v1.6.3 тАФ duplicate bug/issue filing not guarded

**Branch:** `k3d-manager-v1.6.3`
**Files:** `bin/k3dm-webhook`, `Makefile`, `CLAUDE.md`

---

## Problem

Both filing paths write to `docs/bugs/` without checking for existing files. If the same
observation fires twice (or `make file-bug` is called twice with the same title) a second
file is created with a different date prefix but identical slug тАФ resulting in duplicate
bug docs that confuse agents doing future triage.

**Root cause:** `_parse_gemini_observations` constructs `fname` directly and writes it
unconditionally. `make file-bug` also constructs `fname` and writes unconditionally.
Neither path globs for an existing `*-<slug>.md` before creating.

---

## Reproduction

```bash
make file-bug FILE_TITLE="test dupe" FILE_BODY="body"
make file-bug FILE_TITLE="test dupe" FILE_BODY="body"
ls docs/bugs/*test-dupe*   # two files with different date prefixes
```

---

## Fix

### Change 1 тАФ `bin/k3dm-webhook`: skip write if slug already filed

**Exact old block (lines 1604тАУ1617):**

```python
        slug = re.sub(r"[^a-z0-9-]", "", title.lower().replace(" ", "-"))
        today = __import__("datetime").date.today().isoformat()
        fname = Path(REPO_ROOT) / "docs" / "bugs" / f"{today}-{slug}.md"

        except OSError:
            pass
```

**Exact new block:**

```python
        slug = re.sub(r"[^a-z0-9-]", "", title.lower().replace(" ", "-"))
        today = __import__("datetime").date.today().isoformat()
        bugs_dir = Path(REPO_ROOT) / "docs" / "bugs"
        existing = list(bugs_dir.glob(f"*-{slug}.md"))
        if existing:
            filed.append(str(existing[0].relative_to(REPO_ROOT)))
            continue
        fname = bugs_dir / f"{today}-{slug}.md"
        try:
            fname.parent.mkdir(parents=True, exist_ok=True)
            fname.write_text(
                f"# Bug: {title}\n\n"
                f"**Filed:** {today}\n"
                f"**Source:** /ask agent observation\n\n"
                f"## Description\n\n{body}\n"
            )
            filed.append(str(fname.relative_to(REPO_ROOT)))
        except OSError:
            pass
```

**Exact new block:**

```makefile
file-bug: ## FILE_TITLE and FILE_BODY required тАФ write docs/bugs/<date>-<slug>.md
	@test -n "$(FILE_TITLE)" || { echo "Usage: make file-bug FILE_TITLE=<title> FILE_BODY=<body>"; exit 1; }
	@test -n "$(FILE_BODY)"  || { echo "Usage: make file-bug FILE_TITLE=<title> FILE_BODY=<body>"; exit 1; }
	@slug=$$(echo "$(FILE_TITLE)" | tr '[:upper:]' '[:lower:]' | tr ' ' '-' | tr -cd 'a-z0-9-'); \
	 existing=$$(ls docs/bugs/*-$$slug.md 2>/dev/null | head -1); \
	 if [ -n "$$existing" ]; then echo "already filed: $$existing"; exit 0; fi; \
	 fname="docs/bugs/$$(date +%Y-%m-%d)-$$slug.md"; \
	 printf '# Bug: %s\n\n**Filed:** %s\n**Source:** /ask agent observation\n\n## Description\n\n%s\n' \
	   "$(FILE_TITLE)" "$$(date +%Y-%m-%d)" "$(FILE_BODY)" > "$$fname"; \
	 echo "$$fname"
```
