## 1. Upstream Task Reference

This document serves as a pointer to the upstream task spec for migrating the browser automation helper from the legacy `gemini-cli` to the Go-based Google Antigravity CLI (`agy`).

## 2. Consuming Impact & Action Items

Once the refactor is completed and verified upstream in `lib-acg`, the following steps must be taken in `k3d-manager` to integrate the changes:

1. **Subtree Pull:**
   Pull the updated `lib-acg` code into the `scripts/lib/acg/` subtree:
   ```bash
   git subtree pull --prefix scripts/lib/acg \
     git@github.com:wilddog64/lib-acg.git feat/v0.1.8 --squash
   ```
2. **Consuming Plugin Integration:**
   Update `scripts/plugins/gemini.sh` (or `agy.sh` if renamed) to define `_agy_prompt` which wraps `agy --dangerously-skip-permissions` instead of the legacy `gemini` call.
3. **Validation:**
   Run full BATS suite and local Playwright sandbox checks to verify that the Pluralsight and GitHub session checks continue to authenticate successfully under the new CLI wrapper.
