## Required Test Cases

### 1. World-writable dir is rejected

```
Setup: tmp_dir=$(mktemp -d) && chmod 777 "$tmp_dir"
       export PATH="${tmp_dir}:${PATH}"
Call:  _safe_path
Assert: exits non-zero (via _err)
Assert: stderr contains the tmp_dir path
Cleanup: chmod 755 "$tmp_dir" && rmdir "$tmp_dir"
```

### 2. Relative path entry is rejected

```
Setup: export PATH="./bin:${PATH}"
Call:  _safe_path
Assert: exits non-zero
Assert: stderr contains "relative path"
```

### 3. Empty PATH component is rejected

```
Setup: export PATH=":/usr/bin"
Call:  _safe_path
Assert: exits non-zero
Assert: stderr contains "relative path" or "<empty>"
```

### 4. Standard absolute non-writable dirs pass

```
Setup: export PATH="/usr/local/bin:/usr/bin:/bin"
       (these dirs are not world-writable on any standard system)
Call:  _safe_path
Assert: returns 0
```

### 5. Sticky-bit world-writable dir is rejected

```
Setup: tmp_dir=$(mktemp -d) && chmod 1777 "$tmp_dir"
       export PATH="${tmp_dir}:${PATH}"
Call:  _safe_path
Assert: exits non-zero (sticky bit no longer exempts world-writable dirs)
Cleanup: chmod 755 "$tmp_dir" && rmdir "$tmp_dir"
```
