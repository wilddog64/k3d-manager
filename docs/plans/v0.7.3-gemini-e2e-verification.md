## Known Issues to Watch For

- **ArgoCD can't reach Ubuntu k3s**: Verify `argocd cluster list` shows `Ready` not `Unknown`. If Unknown, check that `10.211.55.14:6443` is reachable from the infra cluster.
- **ImagePullBackOff after sync**: ghcr.io packages must be set to `public`. Check: `https://github.com/wilddog64?tab=packages`
- **kustomization.yaml not updated**: The `github-actions[bot]` push in the workflow requires `contents: write` permission. If it fails, check the workflow run logs for the "Update image tag" step.
- **Trivy blocks on CRITICAL/HIGH**: Report the finding — do not bypass `exit-code: 1`. Report back to Claude for a fix decision.
