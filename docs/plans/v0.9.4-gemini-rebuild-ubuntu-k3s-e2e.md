## Step 4 — Merge Kubeconfig on M2 Air

Back on M2 Air (exit Ubuntu SSH session):

```bash
# Copy kubeconfig from Ubuntu VM
scp ubuntu:/etc/rancher/k3s/k3s.yaml ~/.kube/ubuntu-k3s.yaml

# Rename all context/cluster/user entries from 'default' to 'ubuntu-k3s'
sed -i '' 's/: default/: ubuntu-k3s/g' ~/.kube/ubuntu-k3s.yaml

# Point server to tunnel endpoint
sed -i '' 's|server: https://127.0.0.1:6443|server: https://localhost:6443|g' ~/.kube/ubuntu-k3s.yaml

# Merge into main kubeconfig
KUBECONFIG=~/.kube/config:~/.kube/ubuntu-k3s.yaml kubectl config view --flatten > /tmp/merged.yaml
mv /tmp/merged.yaml ~/.kube/config
chmod 600 ~/.kube/config

# Verify
kubectl config get-contexts | grep ubuntu-k3s
kubectl config use-context ubuntu-k3s
kubectl get nodes
```

Expected: `ubuntu` node in `Ready` state.

**STOP — if `kubectl get nodes` returns a credentials error, the kubeconfig is stale.**
Every k3s reinstall generates new certs — the old kubeconfig is invalid. Re-run the
scp + sed + merge steps above to pull the fresh kubeconfig from ubuntu.

**STOP — if node is NotReady after 2 minutes, report `kubectl describe node` output.**

## Step 5 — Re-create ghcr-pull-secret

```bash
export GITHUB_USER=wilddog64
# Retrieve PAT from Vault (infra cluster context)
kubectl config use-context k3d-k3d-cluster
GITHUB_PAT=$(vault kv get -field=token secret/github/pat-read-packages 2>/dev/null)
if [[ -z "$GITHUB_PAT" ]]; then
  echo "PAT not in Vault — provide manually"
  # Do NOT hardcode here — set GITHUB_PAT in your shell session
fi

kubectl config use-context ubuntu-k3s
for ns in apps data payment; do
  kubectl create namespace "shopping-cart-${ns}" --dry-run=client -o yaml | kubectl apply -f -
  kubectl create secret docker-registry ghcr-pull-secret \
    --namespace="shopping-cart-${ns}" \
    --docker-server=ghcr.io \
    --docker-username="${GITHUB_USER}" \
    --docker-password="${GITHUB_PAT}" \
    --dry-run=client -o yaml | kubectl apply -f -
done

# Verify
for ns in apps data payment; do
  kubectl get secret ghcr-pull-secret -n "shopping-cart-${ns}"
done
```

**STOP — if any secret is missing, do not proceed to ArgoCD sync.**
