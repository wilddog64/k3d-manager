# Bugfix: v0.2.0 — GCP credential scoping + acg_restart disconnect TypeError

**Branch:** `fix/stale-cred-restart`
**Files:** `playwright/acg_credentials.js`, `playwright/acg_restart.js`

---

## Problem

**Bug 1 — `_extractGcpCredentials` picks up AWS inputs instead of GCP inputs.**

`credentialsAlreadyVisible` (line 340) uses `page.locator('input[aria-label="Copyable input"]').first()` which matches the first copyable input on the page regardless of provider card. When an AWS sandbox is already running, AWS credentials appear first and `credentialsAlreadyVisible = true`. This skips the Start/Open flow so the GCP panel is never opened. `_extractGcpCredentials` then picks inputs positionally from the whole page and extracts AWS access keys, which fail to parse as GCP service account JSON.

**Root cause:** Neither the `credentialsAlreadyVisible` check nor `_extractGcpCredentials` scope their input queries to the correct provider card.

**Bug 2 — `acg_restart.js` `_cdpBrowser.disconnect().catch()` silently throws TypeError.**

`_cdpBrowser.disconnect().catch(() => {})` at line 296 fails when `disconnect` is not a function on the object returned by `connectOverCDP`. The TypeError is thrown synchronously before `.catch()` can be attached, propagating out of the `restartSandbox` function via the Promise `.catch` handler with `ERROR: _cdpBrowser.disconnect is not a function`.

**Root cause:** `.catch()` only catches rejected promises, not synchronous TypeErrors from calling a non-function property. The `try { await ... } catch {}` pattern used in `acg_credentials.js` handles both.

## Reproduction

```
make credential-test PROVIDER=gcp
# AWS sandbox already running and visible on page
# → INFO: Credentials already populated — skipping Start/Open flow
# → ERROR: Service Account Credentials is not valid JSON
# → ERROR: _cdpBrowser.disconnect is not a function
```

### Change 2 — `playwright/acg_credentials.js`: provider-scoped `credentialsAlreadyVisible` check

**Exact old block:**

```javascript
    const _firstCredInput = page.locator('input[aria-label="Copyable input"]').first();
    const _firstCredVisible = await _firstCredInput.isVisible({ timeout: 3000 }).catch(() => false);
    const _firstCredValue = _firstCredVisible ? await _firstCredInput.inputValue().catch(() => '') : '';
    const credentialsAlreadyVisible = _firstCredVisible && _firstCredValue.trim().length > 0;
```

**Exact new block:**

```javascript
    const _providerInputs = await page.locator('input[aria-label="Copyable input"]').all();
    let _firstCredInCard = null;
    for (const _inp of _providerInputs) {
      const _inCard = await _inp.evaluate((el, label) => {
        let node = el.parentElement;
        for (let _j = 0; _j < 12; _j++) {
          if (!node) break;
          if (new RegExp(label, 'i').test(node.innerText || '')) return true;
          node = node.parentElement;
        }
        return false;
      }, _providerCardLabel).catch(() => false);
      if (_inCard) { _firstCredInCard = _inp; break; }
    }
    const _firstCredVisible = _firstCredInCard
      ? await _firstCredInCard.isVisible({ timeout: 3000 }).catch(() => false)
      : false;
    const _firstCredValue = _firstCredVisible
      ? await _firstCredInCard.inputValue().catch(() => '')
      : '';
    const credentialsAlreadyVisible = _firstCredVisible && _firstCredValue.trim().length > 0;
```

### Change 3 — `playwright/acg_credentials.js`: GCP-card-scoped input collection in `_extractGcpCredentials`

**Exact old block:**

```javascript
  // GCP fields use aria-label="Copyable input" (same as AWS) — getByLabel() finds nothing
  // because the visible labels are not HTML-associated. Use positional extraction.
  const inputs = await page.locator('input[aria-label="Copyable input"]').all();
  console.error(`INFO: Found ${inputs.length} copyable inputs`);

  const username = inputs.length >= 1 ? await inputs[0].inputValue().catch(() => '') : '';
  const password = inputs.length >= 2 ? await inputs[1].inputValue().catch(() => '') : '';
  const serviceAccountJson = inputs.length >= 3 ? await inputs[2].inputValue().catch(() => '') : '';
```

**Exact new block:**

```javascript
  // GCP fields use aria-label="Copyable input". Scope to the Google Cloud sandbox card
  // by walking up the DOM to find an ancestor whose text contains "Google Cloud".
  const _allInputs = await page.locator('input[aria-label="Copyable input"]').all();
  const inputs = [];
  for (const _inp of _allInputs) {
    const _inCard = await _inp.evaluate(el => {
      let node = el.parentElement;
      for (let _i = 0; _i < 12; _i++) {
        if (!node) break;
        if (/google cloud/i.test(node.innerText || '')) return true;
        node = node.parentElement;
      }
      return false;
    }).catch(() => false);
    if (_inCard) inputs.push(_inp);
  }
  console.error(`INFO: Found ${inputs.length} GCP-scoped copyable inputs`);

  if (inputs.length === 0) {
    throw new Error('No GCP credential inputs found in Google Cloud Sandbox card — open the GCP panel and retry');
  }

  const username = inputs.length >= 1 ? await inputs[0].inputValue().catch(() => '') : '';
  const password = inputs.length >= 2 ? await inputs[1].inputValue().catch(() => '') : '';
  const serviceAccountJson = inputs.length >= 3 ? await inputs[2].inputValue().catch(() => '') : '';
```
